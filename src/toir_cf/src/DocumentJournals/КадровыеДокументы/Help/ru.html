Все кадровые документы

Журнал предназначен для работы с документами Прием на работу, Прием на работу списком, Кадровый перевод, Кадровый перевод списком, Увольнение, Увольнение списком.

На форме присутствуют следующие поля для отбора:

  • Сотрудник - сотрудник для отбора.
  • Подразделение - подразделение для отбора.
  • Организация - организация для отбора.

В табличной части Список содержатся следующие поля:

  • Есть файлы - поле, которое показывает наличие прикрепленных файлов к документу.
  • Дата события - дата события, на которое указывает документ.
  • Номер - номер документа.
  • Тип документа - тип документа, расположенного в списке.
  • Сотрудник - сотрудник, на которого оформлен документ.
  • Организация - организация, от лица которой оформляется документ.
  • Комментарий - комментарий к документу.
  • Ответственный - ответственный за текущий документ.

Кнопка [Создать] позволяет создавать документы Прием на работу, Прием на работу списком, Кадровый перевод, Кадровый перевод списком, Увольнение, Увольнение списком.

Кнопка [Создать новый элемент копированием текущего] позволяет создать новый документ, взяв за основу один из существующих.

Кнопка [Печать] позволяет перейти к печатным формам для документов Прием на работу, Прием на работу списком, Кадровый перевод, Кадровый перевод списком, Увольнение, Увольнение списком.

Кнопка [Отчеты] позволяет перейти к формам отчета "Движения документов" и "Связанные документы".
