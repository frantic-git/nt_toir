Справочник «Ставки НДС»

Предназначен для ведения списка ставок НДС.

В форме  выводятся следующие поля:

  • Наименование – наименование ставки.
  • Процент – процент ставки.

Группа «Применение ставки»

Содержит табличную часть с типами налогообложения, для которых доступен выбор данной ставки НДС. По кнопке [Добавить] в табличную часть добавляются типы налогообложения.
