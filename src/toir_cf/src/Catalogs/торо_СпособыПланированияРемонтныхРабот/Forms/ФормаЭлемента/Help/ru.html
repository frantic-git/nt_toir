Справочник «Способы планирования ремонтных работ»

Справочник предназначен для хранения настроек, в соответствии с которыми производится планирование ремонтных работ. Каждая запись содержит совокупность сведений и служит набором параметров для расчета графика ППР.

В форме элемента выводятся следующие поля:

  • Группа – название группы, к которой принадлежит способ планирования.
  • Код – код способа планирования ремонтных работ.
  • Наименование – краткое название способа планирования ремонтных работ.

  • Не учитывать при планировании график работы оборудования – признак, указывающий на то, что при планировании работ по ремонту и обслуживанию оборудования входящего в эту группу с использованием данного способа планирования, не берется в расчет график его работы, указанный на закладке Общие в карточке объекта ремонта.

  • Комментарий – дополнительная информация о способе планирования ремонтных работ.

Кнопка «Настройки»

  • Использовать периодичность – признак расчета ППР в способе по регламентированной стратегии ремонта (раз в период). Включает закладку Периодичность планирования.
  • Учитывать день недели – признак планирования ремонта в определённый день недели. Признак доступен только при включённом флажке Использовать периодичность.
  • Использовать параметры наработки – признак учета наработки в способе планирования при расчете ППР. Включает закладку Параметры наработки.
  • Список параметров наработки – признак устанавливается, если необходимо задать способ планирования по нескольким показателям наработки оборудования. Признак доступен только при включенном флажке Использовать параметры наработки. Ремонт при расчете графика ППР будет запланирован при достижении заданного значения хотя бы одного параметра наработки в списке.

Закладка «Периодичность планирования»

  • Количество периодов – коэффициент, который указывает на количество периодов внутри одного цикла, например, чтобы запланировать выполнение раз в 12 месяцев, необходимо указать периодичность равную месяцу и количество периодов равное двенадцати.
  • Периодичность ремонтных работ – наименование периода, который будет определять длительность цикла проведения ТО и ремонтов; значение выбирается из выпадающего списка и может содержать одно из предопределенных значений: День, Неделя, Декада, Месяц, Квартал, Полугодие, Год.
  • День недели – день недели, на который нужно запланировать выполнение ремонта при расчете графика ППР. Поле доступно, если установлен признак Учитывать день недели.

Закладка «Параметры наработки»

  • Параметр наработки – наименование параметра наработки оборудования, в соответствии с которым будет производится расчет графика ППР.
  • Значение параметра наработки – значение параметра наработки, при достижении которого будет запланировано ТО или ремонт.
