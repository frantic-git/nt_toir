Справочник «Серии номенклатуры»

Справочник предназначен для ведения списка серий номенклатуры. 

Серии идентифицируют экземпляр или партию товаров и наследуются конкретной позицией номенклатуры из вида номенклатуры, который указан для этой позиции. 

Справочник "Серии номенклатуры" доступен по кнопке [Перейти] из справочника "Виды номенклатуры" при включенной функциональной опции "Использовать серии номенклатуры".

В форме выводятся следующие поля:

  • Наименование – название серии.
  • Вид номенклатуры – вид номенклатуры, для которого установлен учет по выбранной серии.
  • Номер – номер серии.
