Справочник «Показатели процессов (KPI)»

Справочник предназначен для ведения списка показателей KPI. 

В форме списка выводятся следующие поля:

  • Наименование – название показателя KPI.
  • Код – код показателя KPI.
  • Тип показателя – аналитический разрез, по которому получаются данные. 
  • Ед. изм. – единицы измерения показателя KPI.
  • Описание – текстовое описание показателя KPI.
  • Вариант заполнения – источник данных для вывода в монитор показателей KPI: формула, схема компоновки данных или внешний показатель.
  • Отчет для расшифровки – название отчета, используемого для расшифровки показателя KPI.
