Справочник «Показатели процессов (KPI)»

Справочник предназначен для ведения списка показателей KPI. 

В форме элемента справочника выводятся следующие поля:

  • Группа – наименование группы показателя KPI
  • Наименование – название показателя KPI.
  • Представление для формул – обозначение показателя KPI, которое будет использоваться в формулах.
  • Ед. изм. – единицы измерения показателя KPI. При установке отметки "Использовать значение разделителя" на место единицы измерения выводится значение разделителя, если в варианте анализа она не заполнена.
  • Разделитель – доступен только для показателей с выбранным вариантом заполнения "СКД". Позволяет из одного набора данных, полученного запросом, выделить несколько наборов, разделенных по значению этого реквизита, и вывести каждый набор как отдельный показатель KPI.
  • Тип показателя – аналитический разрез, по которому получаются данные.
  • Целевой тренд – направление изменений показателя (направление к цели). Выбирается один из возможных вариантов: "Максимизация значения", "Минимизация значения" или "Удержание в допустимом диапазоне".
  • Отчет для расшифровки – название отчета, используемого для расшифровки показателя KPI.
  • Описание – текстовое описание показателя KPI.
  • Формула – доступен для заполнения только для показателей с выбранным вариантом заполнения "Формула". Показатель заполняется по формуле от других показателей.
  • Внешний показатель – доступен для заполнения только для показателей с выбранным вариантом заполнения "Внешний показатель". Данные вводятся в систему с помощью документа "Регистрация значений внешних показателей".
При установке отметки Схема компоновки данных становится активна кнопка для просмотра кода запроса, по которому строится показатель. Редактирование СКД возможно только в режиме запуска "Толстый клиент (управляемое приложение)".
