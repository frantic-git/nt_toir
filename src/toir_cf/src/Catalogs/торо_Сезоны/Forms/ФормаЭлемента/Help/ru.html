Справочник «Сезоны»

Справочник предназначен для ведения списка годовых сезонов выполнения ремонтов.
В форме элемента справочника выводятся следующие поля:
  • Код — код сезона.
  • Наименование —  краткое название сезона.
  • День начала сезона — число месяца начала сезона.
  • Месяц начала сезона — номер месяца начала сезона.
  • День окончания сезона — число месяца окончания сезона.
  • Месяц окончания сезона — номер месяца окончания сезона.
