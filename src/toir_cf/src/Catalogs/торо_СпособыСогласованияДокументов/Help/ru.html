Справочник «Способы согласования документов»

Справочник предназначен для хранения правил, по которым изменяются статусы документов ремонтных работ.

На форме выводятся следующие поля:  

  • Наименование — название способа согласования.
  • Код — код способа согласования.
  • Организация — наименование организации, в которой действует способ согласования.
  • Статус — статус способа согласования: в разработке или действует.
  • Комментарий — текстовый комментарий к способу согласования.
Табличная часть "Матрица перехода" предназначена для задания правил согласования документов и содержит следующие поля:
  • Текущий статус — статус из справочника "Статусы согласования документов ремонтных работ".
  • Этапы согласования — настройки, которые устанавливаются для документа в том или ином статусе.
  • Шаблон уведомления — наименование шаблона письма, которое отправляется в качестве уведомления ответственным за изменение статуса документа пользователям. Поле доступно при включенной ФО "Использовать уведомления о событиях системы" в "Настройка параметров системы - Общие".  
Кнопка [Добавить] предназначена для добавления в способ согласования необходимых статусов из справочника "Статусы согласования документов ремонтных работ".

Закладка «Статус документа»

Предназначена для указания одного или нескольких статусов, в которые можно перевести документ после согласования текущего статуса. Новый статус добавляется по кнопке [Добавить]. Если статус является заключительным, новый статус для него не указывается.

Закладка «Порядок согласования»
  • Согласующие: пользователь — список пользователей, согласующих документ в выделенном статусе.
  • Замещающие: замещающий — список пользователей, которые могут согласовать документ вместо согласующего.

Закладка «Оповещаемые лица»

  • Оповещаемые: оповещаемое лицо — список сотрудников или иных контактных лиц, которые получат оповещение об изменении статуса документа. 
  • E-Mail — адрес электронной почты, на который  будет выслано оповещение об изменении статуса документа
Согласующие, замечающие и оповещаемые лица добавляется с помощью кнопки [Добавить].
