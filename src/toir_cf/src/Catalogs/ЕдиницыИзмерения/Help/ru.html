Справочник «Единицы измерения»

Справочник  предназначен для ведения списка единиц измерения, определенных для позиций номенклатуры.  При необходимости номенклатуре могут быть назначены альтернативные единицы измерения – упаковки, для каждой из которых определяется коэффициент пересчета относительно базовой единицы.

В форме элемента справочника выводятся следующие поля:
  • Код – код единицы измерения.
  • Краткое наименование  – краткое название единицы измерения.
  • Полное наименование – полное наименование единицы измерения. 
  • Международное сокращение – международное сокращение единицы измерения.
  • Тип единицы измерения – дополнительная характеристика единицы измерения: весовая или штучная. 
  • Округлять при расчете статистики – реквизит, указывающий, нужно ли округлять количество номенклатуры с данной единицей измерения.
