Справочник «События уведомлений в документах»

В форме выводятся следующие поля:

  • Наименование - название события. 
  • Код - код элемента справочника.
  • Шаблон сообщения - шаблон сообщения, отправляемого при наступлении события. 
  • Формировать всплывающие уведомления - признак, определяющий необходимость формирования всплывающего уведомления.  
  • Формировать почтовые уведомления - признак, определяющий необходимость формирования сообщения на почту пользователя. 
  • Разделять получателей по организациям - если флаг установлен, то уведомление получат только пользователи с организацией, соответствующей документу.
  • Разделять получателей по подразделениям - если флаг установлен, то уведомление получат только пользователи с подразделением, соответствующим документу. Для выявленных дефектов учитывается реквизит "Подразделение-исполнитель". Во всех остальных случаях используется реквизит "Подразделение".

В табличной части "Оповещаемые" выводятся следующие поля:
  • N - номер строки табличной части.
  • Пользователь - получатель уведомления.
