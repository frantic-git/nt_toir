Справочник «Условия гарантии»

Справочник предназначен для ведения списка условий гарантии.

В форме элемента справочника выводятся следующие поля:
  • Наименование — наименование условия гарантии.
  • Код — код условия гарантии.
  • Проверять наработку — признак, что время действия гарантии определяется наработкой. При выборе признака становятся доступны поля:
    • Показатель наработки — наименование показателя наработки, определяющего период действия гарантии.
    • Значение наработки — значение показателя наработки, в рамках которого действует гарантия.
  • Проверять период гарантии — признак, что время действия гарантии определяется периодом. При установке признака становятся доступны поля:
    • Дата начала — дата начала действия гарантии.
    • Дата окончания — дата окончания действия гарантии.
    • Продолжительность — рассчитывается автоматически после выбора дат начала и окончания действия гарантии, измеряется в днях.
Если установлены обе настройки: и для проверки наработки, и для проверки периода, то объект ремонта будет находиться на гарантии, пока хотя бы одно из условий не будет выполнено.
