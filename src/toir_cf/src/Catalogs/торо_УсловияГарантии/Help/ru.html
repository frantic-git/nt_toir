Справочник «Условия гарантии»

Справочник предназначен для ведения списка условий гарантии.

В форме выводятся следующие поля:

  • Наименование — наименование условия гарантии.
  • Код — код условия гарантии.
  • Проверять наработку — поле, отметка в котором означает, что время действия гарантии определяется наработкой.
  • Показатель наработки — наименование показателя наработки, определяющего период действия гарантии.
  • Значение наработки — значение показателя наработки, по достижении которого гарантия прекращает действовать.
  • Проверять период гарантии — поле, отметка в котором означает, что время действия гарантии определяется периодом.
  • Дата начала — дата начала действия гарантии.
  • Дата окончания — дата окончания действия гарантии.
Если установлены обе настройки: и для проверки наработки, и для проверки периода, то объект ремонта будет находиться на гарантии, пока хотя бы одно из условий не будет выполнено.
