Справочник «Коренные причины дефектов»

Справочник предназначен для хранения списка коренных причин дефектов – причин, по которым возник отказ оборудования. 

В форме списка выводятся следующие поля:

  • Наименование — краткое название коренной причины дефектов.
  • Код — код коренной причины дефектов.
