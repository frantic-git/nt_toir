Справочник «Технологические карты ремонтов»

Справочник предназначен для ведения списка технологических карт. Технологическая карта — это перечень технологических операций, которые необходимо выполнить в рамках того или иного ремонта, и нормативных затрат на их выполнение.

В форме элемента справочника выводятся следующие поля:

  • Группа — наименование группы, в которую входит технологическая карта (может и не заполняться).
  • Наименование — краткое название технологической карты.
  • Код — код технологической карты ремонта.
  • Статус — статус технологической карты: действует она или находится в разработке.
  • Продолжительность — длительность выполнения работ по технологической карте. Поле является обязательным для заполнения. Заполнить продолжительность выполнения работ по технологической карте ремонта можно следующими способами:
    • По нормам времени —  суммируется продолжительность из поля "Норма времени" тех. операций с учетом их количества в тех. карте.
    • По нормам трудозатрат — суммируется продолжительность из поля "Время работы" на закладке "Трудовые затраты" тех. карты с учетом количества затрат.
    • По продолжительности операций/ремонтов — суммируются продолжительности тех. операций из поля "Продолжительность" тех. карты с учетом их количества.
    • Ввести продолжительность — способ ручного ввода продолжительности ремонта. Используется в случае, если технологические операции выполняются параллельно.
  • Тех. карта принята с — дата принятия технологической карты.
  • Выполнение ремонта требует останова оборудования — признак, определяющий необходимость вывода оборудования в простой при выполнении работ по технологической карте.
  • Время простоя — продолжительность простоя при выполнении работ по технологической карте.
  • Вид простоя — наименование вида простоя при выполнении работ по технологической карте.
  • Комментарий — дополнительная текстовая информация о технологической карте ремонта.

Кнопка [Подбор] предназначена для оперативного формирования в левой табличной части списка операций технологической карты с помощью справочника "Технологические операции".

Закладка «Материальные затраты»

  • Номенклатура — наименование позиции номенклатуры, которая входит в перечень материально-технического обеспечения для выбранной ремонтной операции.
  • Ед. изм. — значение из классификатора единиц измерений, в соответствии с которым исчисляется количество позиций номенклатуры.
  • Количество — количество единиц позиций номенклатуры.

Кнопка [Подбор] предназначена для оперативного формирования списка с помощью справочника "Номенклатура".

Закладка «Трудовые затраты»

  • Квалификация — наименование квалификации исполнителя ремонтной операции.
  • Количество — количество исполнителей ремонтной операции.
  • Время работы — длительность выполнения ремонтной операции.

Закладка «Инструменты и техника»

  • Инструмент — наименование инструмента или единицы техники, требуемой для проведения операции.
  • Ед. изм. — обозначение единицы измерения, в которой учитывается инструмент и техника.
  • Количество — количество единиц требуемого инструмента или техники.
  • Расценка — расценка использования инструментов или техники.
  • Валюта — валюта расценки использования инструментов или техники.

Закладка «Измеряемые показатели»
  • Показатель – наименование измеряемого показателя.

Закладка «Меры безопасности»

  • Меры безопасности — название мер безопасности при проведении ремонтных работ

Закладка «Файлы»

  • Файл — наименование прикрепленного к технологической карте файла.
  • Автор — ФИО пользователя, добавившего файл.
  • Изменен — дата изменения файла.
  • Описание — краткое описание прикрепленного файла.
