Справочник «Категории вероятности»

Справочник предназначен для ведения списка категорий вероятности возникновения дефектов.

Кнопка [Загрузить из классификатора] позволяет загрузить данные категорий вероятности из предопределенного списка значений.

В форме списка выводятся следующие поля:

  • Код — код категории вероятности.
  • Наименование — наименование категории вероятности.
  • Описание — текстовое описании категории вероятности.
  • Числовое представление  — соответствующее категории вероятности числовое значение.
