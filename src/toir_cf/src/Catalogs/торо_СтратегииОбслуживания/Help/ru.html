Справочник «Стратегии обслуживания»

Справочник предназначен для ведения списка стратегий обслуживания оборудования. Среди стратегий обслуживания выделяют превентивное обслуживание, аварийное, прогностическое, обслуживание по состоянию, надежности и др. 

В форме списка и форме элемента справочника выводятся следующие поля:

  • Код — код стратегии.
  • Наименование — наименование стратегии.
  • Описание — текстовое описание стратегии.
