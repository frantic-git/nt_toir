Справочник «Виды эксплуатации»

Справочник предназначен для ведения списка видов эксплуатации объектов ремонтов.

В форме элемента справочника выводятся следующие поля:

  • Код — код вида эксплуатации.
  • Наименование — название вида эксплуатации.
  • Тип эксплуатации  — тип эксплуатации объектов ремонта.
