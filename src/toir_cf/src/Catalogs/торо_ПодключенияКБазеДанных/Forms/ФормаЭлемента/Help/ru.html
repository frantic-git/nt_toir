Справочник «Подключения к базе данных»

Справочник предназначен для ведения списка подключений к базе данных при использовании интеграций с АСУ ТП.

В форме элемента справочника выводятся следующие поля:
  • Код — код подключения.
  • Наименование — название подключения.
  • Отключен — признак, при установке которого данные из подключения получаться не будут. 
  • Использовать произвольную строку подключения — используется в случае, когда строка подключения к АСУ ТП отличается от стандартной строки подключения к MS SQL Server. Для указания места, в которое нужно подставить логин и пароль, следует использовать ключевые слова %Логин% и %Пароль% соответственно. При выборе настройке становится доступно поле:
    • Строка соединения — набор обязательных (сервер, имя базы, логин и пароль) и дополнительных (в зависимости от настроек SQL) параметров для подключения к базе данных. 
  • Сервер — адрес сервера, на котором расположена база.
  • База данных — имя базы данных в СУБД.
  • Логин — логин для подключения к базе данных.
  • Пароль — пароль для подключения к базе данных.
  • Период получения данных — частота получения данных (например, раз в 24 часа, раз в час).
  • Текст запроса чтения — текст запроса для чтения загружаемых данных.
  • Использовать пометку данных — признак, при установке которого будет использоваться пометка загружаемых данных.
  • Текст запроса пометки обрабатываемых — текст запроса для пометки обрабатываемых данных.
  • Использовать удаление данных — признак, при установке которого применяется запрос из блока удаления.
  • Текст запроса удаления обработанных — текст запроса для пометки обработанных данных.
  • Использовать разбиение данных на порции — признак, при установке которого записи будут получаться порциями указанного размера с даты прошлого сеанса по текущий момент времени.
  • Размер порции — размер порции данных при использовании разбиения.
