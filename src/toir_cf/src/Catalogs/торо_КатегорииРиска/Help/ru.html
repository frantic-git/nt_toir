Справочник «Категории риска»

Справочник предназначен для ведения списка категорий риска возникновения дефектов.

В форме списка выводятся следующие поля:

  • Код — код категории риска.
  • Наименование — наименование категории риска.
  • Категория вероятности — соответствующая категории риска категория вероятности возникновения дефекта.
  • Категория тяжести последствий — соответствующая категории риска категория тяжести последствий возникновения дефекта.
  • Критичность дефекта — критичность дефекта, соответствующего категории риска.
Кнопка [Загрузить из классификатора] позволяет загрузить данные категорий риска из предопределенного списка значений.

В форме элемента справочника выводятся следующие поля:
  • Наименование — наименование категории риска.
  • Код — код категории риска.
  • Категория вероятности — соответствующая категории риска категория вероятности возникновения дефекта.
  • Критичность дефекта — критичность дефекта, соответствующего категории риска. 
  • Категория тяжести последствий — соответствующая категории риска категория тяжести последствий возникновения дефекта
  • Цвет — цвет, которым категория риска будет обозначаться в матрице оценки рисков.
В полях плановых дат начала и окончания ремонта, а также крайней плановой допустимой даты завершения ремонта определяется вид и количество периодов от даты обнаружения дефекта до наступления соответствующей даты ремонта.
