Справочник «Виды номенклатуры»

Справочник  предназначен для ведения списка видов номенклатуры. Если для отдельной номенклатуры указать вид, то она будет наследовать его настройки типа, использования характеристик и серии. 

Закладка «Основное»

Предназначена для задания основных данных о виде номенклатуры.

Группа «Основное»

  • Группа видов номенклатуры – наименование группы, в которую входит вид номенклатуры.
  • Наименование – название вида номенклатуры.
  • Описание – описание вида номенклатуры.
  • Тип номенклатуры – тип, к которому относится номенклатура данного вида.

Группа «Характеристики»

Отображается, если включена функциональная опция "Использовать характеристики номенклатуры".
  • Использовать характеристики – настройка, при установке которой включается использование характеристик для вида номенклатуры.
  • Использование характеристик – способ использования характеристик: общие для вида номенклатуры, индивидуальные для номенклатуры или не используются.
По кнопке [Перейти] доступен переход в справочник "Характеристики номенклатуры" при условии, что характеристики используются.

Группа «Серии»

Отображается, если включена функциональная опция "Использовать серии номенклатуры".
  • Вести учет по сериям – настройка, при установке которой включается использование серий для вида номенклатуры.
  • Настройка использования серий – выбор способа использования серии
По кнопке [Перейти] доступен переход в справочник "Серии номенклатуры" при условии, что учет по сериям ведется.

Закладка «Доп. реквизиты номенклатуры»


Предназначена для добавления дополнительного реквизита для вида номенклатуры. Дополнительные реквизиты предназначены для описания произвольных свойств вида номенклатуры.

На закладке выводятся следующие поля:
  • Дополнительный реквизит – название дополнительного реквизита вида номенклатуры.

Закладка «Значения по умолчанию»


На закладке можно заполнить значения реквизитов, которые будут подставляться по умолчанию при создании номенклатуры данного вида.

На закладке выводятся следующие поля:
  • Единица хранения – наименование единицы хранения вида номенклатуры, берется из справочника "Единицы измерения".
  • Ставка НДС – величина ставки НДС, которой облагается данный вид номенклатуры.
  • Схема обеспечения – схема обеспечения номенклатуры данного вида; отображается только для вида с типом "Товар" и "Тара".
