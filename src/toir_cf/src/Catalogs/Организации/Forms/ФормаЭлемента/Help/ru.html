Справочник «Организации»

Справочник предназначен для ведения списка организаций.

В форме элемента справочника выводятся следующие поля:
  • Код – код организации.
  • Рабочее наименование – Отображаемое в базе название организации.
  • Полное наименование – Полное название организации.
  • Сокращенное наименование  – краткое название организации.
  • Префикс – символьное значение, которое используется для нумерации документов, регистрируемых в разрезе этой организации (если значение не указано, то нумерация будет сквозная).
  • Юридическое/Физическое лицо – классификация организации по виду. 
  • ИНН – идентификационный номер налогоплательщика.
  • КПП – код причины постановки на учет.  Представляет собой девятизначный цифровой код организации.
  • Код по ОКПО – код по общероссийскому классификатору предприятий и организаций; присваивается в момент регистрации юридического лица и показывает вид деятельности.
  • Юридический адрес – юридический адрес организации.
  • Фактический адрес – фактический адрес организации.
  • Телефон – телефон организации.
  • Факс – факс организации.
  • Электронная почта – электронная почта организации.
  • Почтовый адрес – почтовый адрес организации.
  • Другое – другая дополнительная информация об адресах или способах связи с организацией.
Кнопка [Установить основной] открывает форму выбора сотрудников, в которой выбранная организация может быть установлена для них как основная.
