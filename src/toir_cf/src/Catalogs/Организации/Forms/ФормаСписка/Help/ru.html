Справочник «Организации»

Справочник предназначен для ведения списка организаций.

В форме списка выводятся следующие поля:

  • Код – код организации.
  • Префикс – символьное значение, которое используется для нумерации документов, регистрируемых в разрезе этой организации (если значение не указано, то нумерация будет сквозная).
  • Рабочее наименование – Отображаемое в базе название организации.
  • ИНН – идентификационный номер налогоплательщика.
  • КПП – код причины постановки на учет. Представляет собой девятизначный цифровой код организации.
  • Юридическое / физическое лицо – классификация организации по виду. 
Кнопка [Установить основной] открывает форму выбора сотрудников, в которой выбранная организация может быть установлена для них как основная.
