Справочник «Характеристики номенклатуры»

Справочник предназначен для ведения списка характеристик номенклатуры.

В форме выводятся следующие поля:
  • Рабочее наименование — наименование характеристики номенклатуры.
  • Код — кода характеристики номенклатуры.
  • Наименование для печати — наименование характеристики номенклатуры при выводе на печать.
  • Номенклатура — наименование номенклатуры, для которой вводится характеристика.
