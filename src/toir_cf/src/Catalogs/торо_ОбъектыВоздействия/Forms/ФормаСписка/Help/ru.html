Справочник «Объекты воздействия»

Справочник предназначен для ведения списка объектов, которым возникающие дефекты могут нанести ущерб.

В форме списка выводятся следующие поля:

  • Код — код объекта воздействия.
  • Наименование — наименование объекта воздействия.
  • Описание — текстовое описание объекта воздействия.
Кнопка [Загрузить из классификатора] позволяет загрузить данные объектов воздействия из предопределенного списка значений.
