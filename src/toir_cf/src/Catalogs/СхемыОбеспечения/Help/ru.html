Справочник «Схемы обеспечения»

Справочник предназначен для ведения списка схем обеспечения. Схема обеспечения предназначена для установки основного способа обеспечения потребностей на одном или нескольких складах. 

В форме элемента справочника выводятся следующие поля:

  • Наименование – наименование схемы обеспечения.
  • Используется для товаров – количество товаров, для которых используется схема обеспечения.
  • Склад – наименование склада, на котором производится настройка способа обеспечения.
  • Источник настройки – источник настройки схемы обеспечения для склада.
  • Способ обеспечения потребностей – наименование способа обеспечения потребностей. 
  • Источник обеспечения – наименование контрагента, у которого делается заказ.
  • Срок поставки, дн. – срок покупки: через сколько дней будет получена номенклатура, если заказать ее сегодня.

По кнопке [Заполнить способ обеспечения] открывается форма выбора способа обеспечения.

По кнопке [Очистить способ обеспечения] можно очистить способ обеспечения в выбранной строке. 

По кнопке [Обновить] происходит обновление схемы.
