Справочник «Производители»

Справочник предназначен для ведения списка производителей товаров. Используется для заполнения соответствующего реквизита в справочнике Номенклатура.

В форме  выводятся следующие поля:

  • Наименование – наименование производителя.
  • Группа  – группа, в которую входит производитель; может быть не заполнено.
