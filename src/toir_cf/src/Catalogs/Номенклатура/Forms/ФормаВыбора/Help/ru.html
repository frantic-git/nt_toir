Справочник «Номенклатура»

Справочник предназначен для ведения списка номенклатуры.

В форме выбора выводятся следующие поля:
  • Поиск – поле для поиска по коду, артикулу или наименованию номенклатуры. 
  • По точному соответствию – настройка, при выборе которой будет идти поиск по точному соответствию.
  • Фильтр – настройка способа отбора номенклатуры: по видам или по иерархии.
Кнопка [Карточка номенклатуры] открывает печатную форму карточки выбранной номенклатуры. 

Табличная часть «Список номенклатуры»

Содержит список позиций номенклатуры, принадлежащих группе или виду номенклатуры, в зависимости от выбранного фильтра. 
  • Номенклатура – наименование номенклатуры.
  • Код – код номенклатуры.
