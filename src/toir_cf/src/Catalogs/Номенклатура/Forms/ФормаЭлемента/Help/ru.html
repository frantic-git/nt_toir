Справочник «Номенклатура»

Справочник предназначен для ведения списка номенклатуры.

В форме элемента справочника выводятся следующие поля:
  • Рабочее наименование – наименование контрагента, которое будет использоваться для быстрой идентификации в программе.
  • Полное наименование – полное наименование номенклатуры.
  • Артикул – артикул номенклатуры.
  • Код – код номенклатуры в системе.
Переключатель [Показать все/Свернуть] все позволяет развернуть или свернуть все группы. 

Группа «Основные параметры учета»

Предназначена для задания учетной информации номенклатуры.
  • Группа списка – наименование группы в иерархии, в которую входит номенклатура.
  • Вид номенклатуры – вид номенклатура: товар, тара, услуга или работа.
  • Тип номенклатуры – тип номенклатуры: наследуется из вида номенклатуры. 
  • Характеристики – характеристика номенклатуры; отображается, если включена функциональная опция "Использовать характеристики номенклатуры", для вида номенклатуры установлена настройка "Использовать характеристики" и заполнена настройка использования характеристик. 
  • Серии – серия номенклатуры; отображается, если включена функциональная опция "Использовать серии номенклатуры", для вида номенклатуры установлена настройка "Вести учет по сериям" и заполнена настройка использования серий. 
  • Ставка НДС – ставка НДС в цене номенклатуры. 

Группа «Единицы измерения»

Предназначена для указания единиц измерения номенклатуры. 
  • Единица хранения – единица измерения номенклатуры, в которой происходит ее движение в системе (указанное значение по умолчанию подставляется в документы).

Группа «Обеспечение и производство»

Предназначена для выбора схемы обеспечения номенклатуры. 

  • Схема обеспечения – наименование схемы обеспечения, по которой обеспечивается потребность в номенклатуре на складе. У номенклатуры с типом "Товар" и "Тара" схема обеспечения наследуется от вида номенклатуры.

Гиперссылка "Настройка способов обеспечения потребностей" открывает одноименную обработку. У номенклатуры видов "Работа" и "Услуга" в форме элемента справочника группа "Обеспечение и производство" отсутствует.

Группа «Описание»

Предназначена для ввода текстовой и визуальной информации о номенклатуре.
  • Изображение – поле для загрузки изображения номенклатуры.
  • Текстовое описание – поле для текстового описания номенклатуры.

Группа «Сведения о производителе»

Предназначена для ввода сведений о производителе номенклатуры.
  • Производитель – наименование производителя из справочника "Производители".
Кнопка [Карточка номенклатуры] открывает печатную форму карточки выбранной номенклатуры.
