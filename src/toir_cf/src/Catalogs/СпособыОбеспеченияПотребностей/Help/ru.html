Справочник «Способы обеспечения потребностей»

Справочник предназначен для ведения списка способов обеспечения потребностей.

В форме списка справочника выводятся следующие поля:

  • Наименование – наименование способа обеспечения.
  • Источник обеспечения потребностей – наименование контрагента, у которого делается заказ.
  • Срок исполнения заказа – срок покупки: через сколько дней будет получена номенклатура, если заказать ее сегодня. Указывается в рабочих днях по производственному календарю РФ.

В форме элемента справочника выводятся следующие поля:

  • Наименование – наименование схемы обеспечения.
  • Использовать для работы: с несколькими поставщиками – отметка, при выборе которой параметры планирования способа обеспечения применяются для заказов различным поставщикам.
  • Использовать для работы: с одним поставщиком – отметка, при выборе которой параметры планирования способа обеспечения применяются для заказа только одному поставщику.
  • Срок покупки – срок исполнения заказа на покупку, определяет дату ближайшего возможного поступления товаров.
