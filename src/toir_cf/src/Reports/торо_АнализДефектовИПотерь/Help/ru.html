Отчет «Анализ дефектов и потерь»

В данном отчете за определенный период и с указанной детализацией выводится информация о количестве выявленных дефектов, устраненных, а также среднем и суммарном времени их устранения. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет выявленных дефектов оборудования.

Открывается по команде Анализ дефектов и потерь: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - отбор дефектов по дате регистрации документа "Выявленные дефекты";
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года;
  • Подразделение исполнитель - отбор, позволяющий отобразить дефекты, выполняемые определенным подразделением (поле "Подразделение-исполнитель" документа "Выявленные дефекты").

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианты отчета:
  1. Анализ дефектов и потерь: таблица - открывается по умолчанию. Данные о дефектах сгруппированы по виду дефекта.
    В этом варианте отчета по умолчанию в строках выводится информация о зарегистрированных в системе дефектах объектов ремонта. Регистрация дефектов объектов ремонта выполняется документом "Выявленные дефекты":
    • Вид дефекта - наименование вида дефекта объекта ремонта;
    • Количество дефектов - количество введенных в систему документов "Выявленные дефекты" с определенным видом дефекта (поле "Вид дефекта"):
      • Выявлено - количество введенных в систему документов, имеющих статусы "Зарегистрирован", "Запланировано устранение", "Устраняется";
      • Устранено - количество устраненных дефектов (статус документа "Устранен". Появляется после введения документа "Акт о выполнении этапа работ"  (с завершающим флагом));
    • Время ремонтов - количество времени, затраченное на устранение дефекта:
      • Среднее - среднее время устранения дефекта (рассчитывается по формуле: Выявленный дефект 1 + Выявленный дефект 2 + Выявленный дефект n / N, где N - количество устраненных дефектов);
      • Суммарное - суммарное время устранения дефекта за указанный период отчета по каждому виду дефекта;
    • Итоговая колонка - итоговые данные о количестве дефектов и времени ремонтов за весь указанный период отчета.
  2. Анализ дефектов и потерь: диаграмма - отчет предназначен для сравнительного анализа выявленных и устраненных дефектов в зависимости от вида дефекта в графическом виде. А также, для анализа длительности ремонтов разных видов дефекта.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
