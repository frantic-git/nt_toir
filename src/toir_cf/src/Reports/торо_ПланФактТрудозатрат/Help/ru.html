Отчет «План-фактный анализ трудозатрат»

Отчет предназначен для план-фактного анализа трудозатрат по ремонтным работам. Отчет содержит перечень объектов ремонта предприятия, по которым выполнялись ремонты за указанный период. Для каждого объекта ремонта перечисляются выполненные технологические операции, перечень квалификаций и соответствующие им количества задействованных в работах сотрудников и сумма трудозатрат в разрезе по плану, указанному в заявках на ремонт, и факту, заданному в актах выполненных работ.

Открывается по команде План-фактный анализ трудозатрат:

  • в списке Отчеты по управлению персоналом подсистемы Управление персоналом;
  • в списке Отчеты по нарядам и работам подсистемы Наряды и работы.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета — отбор трудозатрат по плановым датам;
  • Структура иерархии — отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии");
  • Учитывать план графики ППР — настройка, позволяющая учитывать трудозатраты по ремонтам из документов "План-график ППР";
  • Учитывать сметы (заявки на ремонт) по дефектам — настройка, позволяющая учитывать трудозатраты по ремонтам из документов "Выявленные дефекты";
  • Учитывать сметы (заявки на ремонт) по внешним основаниям — настройка, позволяющая учитывать трудозатраты по ремонтам из документов "Внешнее основания для работ";
  • Учитывать регламентные мероприятия — настройка, позволяющая учитывать трудозатраты по ремонтам из документов  "График регламентных мероприятий";
  • Вид документа — настройка, позволяющая отображать трудовые затраты по виду документа.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:
  1. План-фактный анализ трудозатрат — открывается по умолчанию. Данные о трудовых затратах сгруппированы по объекту ремонта. В этом варианте отчета по умолчанию в строках выводится информация о фактических и плановых трудозатратах. Фиксация фактических затрат выполняется документами "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия", плановых - документом "Смета ремонта (Заявка на ремонт)":
    • Объект ремонта — наименование объекта ремонта;
    • Ремонтная работа — наименование технологической операции, для выполнения которой требуются трудовые затраты (поле "Ремонтная работа" из дерева ремонтных работ документов "Смета ремонта (Заявка на ремонт)", "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия");
    • Квалификация — название квалификации ремонтного персонала (поле "Квалификация" из документов "Смета ремонта (Заявка на ремонт)", "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Исполнители");
    • Количество сотрудников — количество, которое необходимо на выполнение ремонтной работы:
      • План — регистрируется документом "Смета ремонта (Заявка на ремонт)" (поле "Количество" на вкладке "Исполнители");
      • Факт — регистрируется документами "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" (поле "Количество" на закладке "Исполнители");
    • Сумма — сумма затрат на трудовые ресурсы, которая потребовалась для выполнения ремонтной работы:
      • План — регистрируется документом "Установка стоимостей квалификаций"(рассчитывается по формуле: Стоимость часа* Время работы*Количество сотрудников);
      • Факт — регистрируется документами "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" (поле "Стоимость работ" на закладке "Исполнители");
    • Время — время, затраченное трудовыми ресурсами на выполнение ремонтной работы:
      • План — регистрируется документом "Смета ремонта (Заявка на ремонт)" (поле "Время работы" на вкладке "Исполнители");
      • Факт — регистрируется документами "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" (поле "Время работы" на закладке "Исполнители");
    • Отклонение — рассчитывается как разность между полями плана (количества, суммы, времени) и факта (количества, суммы, времени). 
  2. План-фактный анализ трудозатрат (с иерархией) — вариант отчета, в котором трудовые затраты ремонтов сгруппированы по объекту иерархии, объекту ремонта:
    • Объект иерархии — наименования всех родителей объекта ремонта (объекты ремонта или группы объектов ремонта).
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
