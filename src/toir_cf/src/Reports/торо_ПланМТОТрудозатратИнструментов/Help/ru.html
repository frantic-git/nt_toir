Отчет «План использования МТО, трудозатрат и инструментов»

Отчет предназначен для получения информации об использовании материальных и трудовых затрат, а также, затрат на инструменты и технику в плановых ремонтных работах.  
Открывается по команде План использования инструментов и техники: 
  • в списке Отчеты по планированию ТОиР подсистемы Планирование ТОиР;
  • в списке Отчеты по МТО подсистемы МТО.
Параметры отчета
Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:
  • Период отчета - отбор ремонтов по плановым датам;
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года;
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии");
  • Учитывать план графики ППР - настройка, позволяющая учитывать ремонты из документов "План-график ППР";
  • Учитывать регламентные мероприятия - настройка, позволяющая учитывать ремонты из документов  "График регламентных мероприятий";
  • Учитывать Сметы ремонта (Заявки на ремонт) по дефектам - настройка, позволяющая учитывать ремонты из документов "Выявленные дефекты";
  • Учитывать Сметы ремонта (Заявки на ремонт) по внешним основаниям - настройка, позволяющая учитывать ремонты из документов "Внешнее основания для работ".
Варианты отчета
С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны пять предопределенных варианта отчета:
  1. План использования инструментов и техники: таблица - открывается по умолчанию. В этом варианте отчета выводятся плановые затраты на инструменты и технику. Данные группируются по используемым в ремонтах инструментам. Также в разрезе инструментов выводятся данные об объектах ремонта, виде ремонта и технологической карте.
    • Инструмент - наименование инструмента, который используется для выполнения ремонтной работы (поле "Инструмент" из документа "Смета ремонта (Заявка на ремонт)" на закладке "Инструменты и техника" для документов "Внешнее основание для работ", "Выявленные дефекты"; поле "Инструмент" согласно "Технологической карте" из справочника "Объекты ремонта" на вкладке "Нормативы планирования - Нормативные ремонты" для документа "План-график ППР";  поле "Инструмент" согласно "Технологической карте" из справочника "Списки объектов регламентных мероприятий" на вкладке "Регламентные мероприятия" для документа "График регламентных мероприятий");
    • Объект ремонта - наименование объекта ремонта;
    • Вид ремонта - вид ремонта (поле "Вид ремонта" из документа "План-график ППР"; поле "Вид мероприятия" из документа "График регламентных мероприятий"; поля "Вид ремонта при вводе на основании выявленных дефектов" и "Вид ремонта при вводе на основании внешних оснований" из настроек параметров подсистемы ТОИР);
    • Нормативный ремонт - наименование технологической карты ремонта (ТЧ "Ремонтная работа" из документа "Смета ремонта (Заявка на ремонт)" на закладке "Инструменты и техника" для документов "Внешнее основание для работ", "Выявленные дефекты"; поле "Технологическая карта" из справочника "Объекты ремонта "на вкладке "Нормативы планирования - Нормативные ремонты" для документа "План-график ППР"; поле "Технологическая карта" из справочника "Списки объектов регламентных мероприятий" на вкладке "Регламентные мероприятия" для документа "График регламентных мероприятий");
    • Максимальная потребность - количество, которое необходимо на выполнение ремонтной работы (регистрируется документом "Смета ремонта (Заявка на ремонт)" (поле "Количество" на вкладке "Инструменты и техника").
  2. План использования инструментов и техники: диаграмма - вариант отчета, в котором плановые затраты на инструменты и технику отображаются в разрезе инструментов в графическом виде по всем ремонтным работам за выбранный период.
  3. План МТО: таблица - вариант отчета, в котором выводят данные о материальных затратах. Отчет содержит перечень объектов ремонта предприятия, по которым выполнялись ремонты за указанный период. Для каждого объекта ремонта выводится вид ремонта. В этом варианте отчета не рассчитываются итоги по полю "Количество", т.к. в нем выводят значения в разных единицах измерения:
    • Объект ремонта - наименование объекта ремонта;
    • Вид ремонта - вид ремонта (поле "Вид ремонта" из документа "План-график ППР"; поле "Вид мероприятия" из документа "График регламентных мероприятий"; поля "Вид ремонта при вводе на основании выявленных дефектов" и "Вид ремонта при вводе на основании внешних оснований" из настроек параметров подсистемы ТОИР);
    • Номенклатура - наименование номенклатуры, которая используется для выполнения ремонтной работы (поле "Номенклатура" из документа "Смета ремонта (Заявка на ремонт)" на закладке "Материальные затраты" и Запчасти" для документов "Внешнее основание для работ", "Выявленные дефекты"; поле "Номенклатура" согласно полю "Технологическая карта" из справочника "Объекты ремонта" на вкладке "Нормативы планирования - Нормативные ремонты/Запчасти" для документов "План-график ППР", "График регламентных мероприятий");
    • Характеристика - наименование характеристики номенклатуры (поле "Характеристика" из документа "Смета ремонта (Заявка на ремонт)" для документов "Внешнее основание для работ", "Выявленные дефекты"; поле "Характеристика" согласно полю "Технологическая карта" из справочника "Объекты ремонта" для документов "План-график ППР", "График регламентных мероприятий"). Выводится, если установлена настройка системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
    • Количество - количество, которое запланировано для выполнения ремонтной работы (поле "Количество" из документа "Смета ремонта (Заявка на ремонт) на закладках "Материальные затраты", "Запчасти" для документов "Внешнее основание для работ", "Выявленные дефекты"; поле "Количество" согласно полю "Технологическая карта" из справочника "Объекты ремонта" для документов "План-график ППР", "График регламентных мероприятий");
    • Сумма - сумма плановых затрат на материальные ресурсы, которая потребуется для выполнения ремонтной работы (рассчитывается по формуле: Плановое количество * Цена на плановую дату ремонта. Цена номенклатуры устанавливается документом "Установка цен номенклатуры");
    • Итоговое количество - итоговое количество, которое планируется затратить на выполнение ремонтной работы за указанный период;
    • Итоговая сумма - итоговая сумма плановых затрат на материальные ресурсы, запчасти, требующихся для выполнения ремонтной работы за указанный период.
  4. Плановая занятость сотрудников: таблица - вариант отчета, в котором выводится информация о плановой занятости сотрудников. Отчет содержит перечень квалификаций сотрудников, которыми выполнялись ремонты за указанный период. Для каждой квалификации перечисляются объекты ремонта и виды ремонта, а также, потребность и время в квалификации. 
    • Квалификация - название квалификации ремонтного персонала (поле "Квалификация" из документов "Смета ремонта (Заявка на ремонт)", "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Исполнители");
    • Объект ремонта - наименование объекта ремонта; 
    • Вид ремонта - вид ремонта (поле "Вид ремонта" из документа "План-график ППР"; поле "Вид мероприятия" из документа "График регламентных мероприятий"; поля "Вид ремонта при вводе на основании выявленных дефектов" и "Вид ремонта при вводе на основании внешних оснований" из настроек параметров подсистемы ТОИР); 
    • Максимальная потребность - количество, которое необходимо на выполнение ремонтной работы (регистрируется документом "Смета ремонта (Заявка на ремонт)" (поле "Количество" на вкладке "Исполнители"); 
    • Время работы - время, которое планируется затратить трудовыми ресурсами на выполнение ремонтной работы (регистрируется документом "Смета ремонта (Заявка на ремонт)" (поле "Время работы" на вкладке "Исполнители").
  5. Плановая занятость сотрудников: диаграмма - вариант отчета, в котором плановая занятость сотрудников отображается в разрезе квалификаций в графическом виде по всем ремонтным работам за выбранный период.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
