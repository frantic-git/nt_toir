Отчет «Остановочные ремонты»

Отчет предназначен для получения информации об остановочных ремонтах за период и проценте их завершения.
Открывается по команде Остановочные ремонты: : в списке Отчеты по планированию ТОиР в подсистеме Планирование ТОиР.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - фильтр по плановому периоду основного ремонта. Действие отбора не распространяется на связанные ремонты. Отбор работает так, что хотя бы часть ремонта попадала в период отчета;
  • Показывать завершенные ремонты - настройка, отвечающая за вывод завершенных ремонтов (по которым введен акт с признаком завершения этого ремонта);
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Остановочные ремонты. Данные сгруппированы по объектам ремонта. 
В этом варианте отчета на втором уровне выводятся основные ремонты объекта ремонта, а на третьем - все зависимые от основного ремонты.

Основные поля отчета:

  • Объект и вид ремонта - ссылки на объект и вид ремонта, по которым зарегистрированы остановочный и зависимые ремонты;
  • Даты начала и окончания - актуальные плановые даты ремонта;
  • Статус ремонта - текущий статус ремонта;
  • Завершен - признак завершения ремонта;
  • Процент завершения - процент завершения остановочных ремонтов, вычисляемый по формуле: 100% * Количество выполненных ремонтов / Общее количество ремонтов, где Количество выполненных ремонтов - количество выполненных ремонтов в пределах одного остановочного, Общее количество ремонтов - число всех ремонтов в этом же остановочном ремонте. При вычислении числителя и знаменателя на уровне объекта ремонта и основного ремонта учитывается также и сам остановочный ремонт.
При расшифровке объекта и вида ремонта открываются соответствующие значения, при расшифровке остальных полей - регистратор основного или зависимого ремонта (в зависимости от уровня).

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
