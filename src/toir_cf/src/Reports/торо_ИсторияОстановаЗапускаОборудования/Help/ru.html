Отчет «Останов/запуск оборудования»

Отчет предназначен для получения информации о датах остановов и запусков, а также причинах остановов объектов ремонта за период. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет состояния оборудования.
Открывается по команде Останов/ запуск оборудования: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду регистрации остановов и запусков объектов ремонта в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:

  1. Останов/запуск оборудования: таблица - открывается по умолчанию. Данные об остановах и запусках объектов ремонта (о дате и времени) сгруппированы по объектам ремонта и по документам, которыми зарегистрировано изменение состояния объекта ремонта. Информация по остановам (запускам) объекта ремонта выводится только в случае, если дата останова или запуска попадает в указанный период.
    В этом варианте отчета по умолчанию в строках выводится информация о зарегистрированных в системе датах останова и запуска объектов ремонта. Регистрация остановов и запусков объектов ремонта выполняется документами "Состояния объектов ремонта" и "Ввод начальных данных":
    • Объект ремонта - наименование объекта ремонта;
    • Документ - ссылка на документ "Состояния объектов ремонта" или на документ "Ввод начальных данных", в которых зарегистрирован останов или запуск объекта ремонта;
    • Вид операции - вид операции документа "Состояния объектов ремонта", в котором зарегистрирован останов или запуск объекта ремонта.
      У документа "Состояния объектов ремонта" 2 вида операции:
      1. Состояние на период;
      2. Состояние на дату.
      У документа "Ввод начальных данных" 2 вида операции:
      1. По объекту ремонта;
      2. По списку объектов ремонта (не используется в отчетах).
    • Причина простоя - причина простоя объекта ремонта, которая указывается при регистрации вида эксплуатации с типом "Простой" объекта ремонта;
    • Остановлено* - дата и время регистрации вида эксплуатации с типом "Простой" объекта ремонта;
    • Запущено* - дата и время регистрации вида эксплуатации с типом эксплуатации, закрывающим "Простой" объекта ремонта;
    Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
  2. Останов/запуск оборудования: диаграмма - отчет предназначен для получения информации о времени нахождения в простое объектов ремонта (в секундах) в разрезе причин простоя, выраженном в процентах от общего времени нахождения в простое за выбранный период, в графическом виде.
Примечания*:
В отчете имеется ряд особенностей, связанный с группировкой "Объект ремонта - Документ - Вид операции":
  • Если состояние объекта ремонта регистрируется документом "Ввод начальных данных", то в отчете в одной строке всегда будет заполнено только одно поле "Остановлено" или "Запущено" в зависимости от типа эксплуатации для объекта ремонта;
  • Если в пределах одного документа "Состояния объектов ремонта" имеется незавершенный "Простой" объекта ремонта (т.е. не зарегистрированы другие состояния, отличные от простоя, по объекту ремонту на более поздний период), в отчете в одной строке всегда будет заполнена только дата останова объекта ремонта;
  • Если по объекту ремонта было зарегистрировано состояние "Простой" (но не закрыто в рамках одного документа), то при оформлении состояния другим документом по объекту ремонта, отличного от простоя, в отчете в одной строке всегда будет заполнена только дата запуска объекта ремонта.
