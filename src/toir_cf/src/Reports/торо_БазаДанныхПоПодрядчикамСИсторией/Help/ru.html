Отчет «База данных по подрядчикам с историей»

Отчет предназначен для получения информации об оценках работы подрядчиков, привлеченных для выполнения ремонтов. Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров системы - Интерфейс и ввод документов - Планирование ТОиР: Использовать лоты.
Открывается по команде База данных по подрядчикам с историей: в списке Отчеты по нарядам и работам подсистемы Наряды и работы.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - База данных по подрядчикам с историей. Данные об оценках работ подрядчиками сгруппированы по документам, в которых зафиксировано выполнение работ подрядчиком (т.е. по документам "Акт о выполнении этапа работ" ), подрядчикам, датам и видам оценки.
В этом варианте отчета по умолчанию в строках выводится информация о критериях и значении оценки работ, выполненных подрядчиками. Выполнение ремонтов объектов ремонта фиксируется документами "Акт о выполнении этапа работ" , а регистрация оценок работ - документами "Оценка работы подрядчика":

  • Акт - ссылка на документ "Акт о выполнении этапа работ", в котором ремонт выполнялся подрядчиком;
  • Подрядчик - подрядчик, выполняющий работы (поле "Исполнитель" с закладки "Исполнители/По ремонту" документа "Акт о выполнении этапа работ");
  • Дата - дата оценивания работ подрядчика (поле "от:" из документа "Оценка работы подрядчика");
  • Вид оценки - наименование вида оценки работы подрядчика (поле "Вид оценки" из документа "Оценка работы подрядчика");
  • Критерий оценки - наименование критерия оценки работы подрядчика (поле "Критерий оценки" из документа "Оценка работы подрядчика");
  • Оценка - оценка работы подрядчика по указанному виду оценки (поле "Оценка" из документа "Оценка работы подрядчика");
  • Значение - значение критерия оценки работы подрядчика (поле "Значение" из документа "Оценка работы подрядчика").
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
