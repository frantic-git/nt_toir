Отчет «Журнал корректирующих мероприятий»

Отчет предназначен для получения информации о состояниях корректирующих мероприятий. Доступен при включенном флаге в разделе Настройка параметров системы – Показатели эксплуатации: Использовать анализ причин дефектов. 

Открывается по команде Журнал корректирующих мероприятий: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - отбор по сроку выполнения корректирующего мероприятия.

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный варианта отчета – Журнал корректирующих мероприятий. Данные о мероприятиях сгруппированы по подразделениям. Фиксация корректирующих мероприятий выполняется документом "Анализ причин дефекта":

  • Подразделение - наименование подразделения организации, проводящего анализ причины дефекта;
  • Анализ причин дефекта - ссылка на документ "Анализ причин дефекта";
  • № - порядковый номер мероприятия в группировке;
  • Описание мероприятия - поле "Описание" из документа "Анализ причин дефекта";
  • Выявленный дефект - ссылка на документ "Выявленные дефекты";
  • Ответственный - ФИО пользователя/сотрудника, который является ответственным выполнение корректирующего мероприятия;
  • Объект ремонта - наименование объекта ремонта;
  • Срок выполнения - срок выполнения корректирующего мероприятия;
  • Отказавший элемент - поле "Отказавший элемент" из документа "Выявленный дефект";
  • Внешние основание/ задание - ссылка на документ  "Внешнее основания для работ" или "Задание";
  • Краткое описание дефекта - поле "Краткое описание" со вкладки "Описание события" документа "Анализ причин дефекта";
  • Статус - статус документа "Внешнее основание для работ" (Зарегистрирован, Запланировано устранение, В  работе, Выполнен, Отклонен) или статус "Задания" (Не завершен, Завершен);
  • Дата выполнения - фактическая дата выполнения корректирующего мероприятия;
  • Выполнено - количество выполненных корректирующих мероприятий в рамках группировки;
  • Количество - количество корректирующих мероприятий в рамках группировки.

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
