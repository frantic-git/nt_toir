Отчет «Принятие к учету, выбытие оборудования»

Отчет предназначен для получения информации о датах принятия к учету объектов ремонта, используемых на предприятии, и датах их выбытия. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Оборудование и нормативы: Использовать документы принятия и списания оборудования.
Открывается по команде Принятие к учету, выбытие оборудования: в списке Отчеты по оборудованию и нормативам подсистемы Оборудование и нормативы.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по датам принятия и выбытия объектов ремонта в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны три предопределенных варианта отчета:

  1. Выбытие оборудования: таблица - отчет предназначен для получения информации о снятых с учета объектах ремонта.
    В этом варианте отчета по умолчанию в строках выводится информация о снятых с учета объектах ремонта, датах их выбытия и документах, в которых было отражено списание объектов ремонта с учета. Процесс выбытия объектов ремонта выполняется документом "Выбытие объекта ремонта":
    • Объект ремонта - наименование объекта ремонта, снятого с учета;
    • Дата снятия с учета - дата снятия объекта ремонта с учета (поле "Дата списания" из документа "Выбытие объекта ремонта");
    • Документ снятия с учета - ссылка на документ "Выбытие объекта ремонта".
  2. Принятие к учету, выбытие оборудования: таблица - открывается по умолчанию. В этом варианте отчета по умолчанию в строках выводится информация о результатах принятия и списания объектов ремонта с учета (даты принятия и списания объектов ремонта с учета и документы, отражающие это). Процесс принятия к учету объектов ремонта выполняется документом "Принятие объекта ремонта к учету", а процесс выбытия объектов ремонта - документом "Выбытие объекта ремонта":
    • Объект ремонта - наименование объекта ремонта;
    • Дата принятия - дата принятия объекта ремонта к учету (поле "Дата ввода в эксплуатацию" из документа "Принятие объекта ремонта к учету");
    • Документ принятия - ссылка на документ "Принятие объекта ремонта к учету";
    • Дата снятия с учета - дата снятия объекта ремонта с учета (поле "Дата списания" из документа "Выбытие объекта ремонта");
    • Документ снятия с учета - ссылка на документ "Выбытие объекта ремонта".
  3. Принятие к учету: таблица - отчет предназначен для получения информации о принятых к учету объектах ремонта.
    В этом варианте отчета по умолчанию в строках выводится информация о принятых к учету объектах ремонта, датах их принятия и документах, в которых было отражено принятие объектов ремонта к учету. Процесс принятия к учету объектов ремонта выполняется документом "Принятие объекта ремонта к учету":
    • Объект ремонта - наименование объекта ремонта, принятого к учету;
    • Дата принятия - дата принятия объекта ремонта к учету (поле "Дата ввода в эксплуатацию" из документа "Принятие объекта ремонта к учету");
    • Документ принятия - ссылка на документ "Принятие объекта ремонта к учету".
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
