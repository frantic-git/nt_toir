Отчет «Запланированные ремонты объектов ремонта»

Отчет предназначен для получения информации о том, для каких объектов ремонта введены документы "План-график ППР" на выбранный период, а для каких нет. Если документ не введен, то в поле "Документ" будет пусто. Для доступа к отчету необходимо установить настройку в разделе Настройка параметров системы – Интерфейс и ввод документов – Планирование ТОиР: Использовать ППР. 

Отчет открывается по команде Запланированные ремонты объектов ремонта в списке Отчеты по планированию ТОИР подсистемы Планирование ТОиР.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета — отбор запланированных ремонтов по датам планирования графиков ППР;
  • Структура иерархии — отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии"). 

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один вариант отчета — Запланированные ремонты объектов ремонта. Данные о запланированных ремонтах группируются по подразделениям-владельцам объектов ремонта в разрезе объектов ремонта и видов ремонтных работ.

  • Подразделение — подразделение-владелец объекта ремонта;
  • Объект ремонта — наименование объекта ремонта;
  • Вид ремонта — наименование вида ремонтных работ (поле "Вид ремонта" из документа "План-график ППР" для соответствующего объекта ремонта). 
  • Документ — ссылка на документ "План-график ППР";
  • Дата планирования — дата, определяющая начало периода формирования графика ППР (поле "Дата планирования" из документа "План-график ППР").
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
