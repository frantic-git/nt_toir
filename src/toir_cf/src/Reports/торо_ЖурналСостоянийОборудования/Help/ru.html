Отчет «Журнал состояний оборудования»

Отчет предназначен для получения информации о времени нахождения объекта ремонта в состоянии (простой, эксплуатация, испытания) за указанный период. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет состояния оборудования.
Открывается по команде Журнал состояний оборудования: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду действия простоев объектов ремонта в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Журнал состояний оборудования. Данные сгруппированы по объектам ремонтам, также, выводится информация о текущем состоянии оборудования, причине простое, времени нахождения в состоянии (простой, эксплуатация, испытания):
  • Объект иерархии - объект ремонта или группа объектов ремонта, имеющие подчиненные объекты, т.е. родитель объекта ремонта;
  • Объект ремонта - наименование объекта ремонта;
  • Дата начала - дата начала изменения состояния объекта ремонта;
  • Дата окончания - дата окончания изменения состояния объекта ремонта;
  • Состояние - состояние объекта ремонта на момент формирования отчета;
  • Причина простоя - причина простоя объекта ремонта (заполняется только для состояния с типом эксплуатации "Простой");
  • Время простоя/эксплуатации/на испытаниях - время нахождения объекта ремонта в состоянии;
  • Итого - итоговая сумма нахождения объекта ремонта во всех состояниях (суммируется время нахождения в простое, эксплуатации и на испытаниях).
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
