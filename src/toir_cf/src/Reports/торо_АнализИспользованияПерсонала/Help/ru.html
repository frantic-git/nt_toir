Отчет «Анализ использования персонала»

Отчет предназначен для анализа использования персонала по ремонтным работам в разрезе конкретных сотрудников. В отчете выводится информация о плановом и фактическом времени выполнения работ сотрудником. Плановое время работ учитывается согласно графику работы сотрудника, фактическое - согласно затраченному времени в рамках конкретного вида ремонта за указанный период.
Открывается по команде Анализ использования персонала:

  • в списке Отчеты по управлению персоналом подсистемы Управление персоналом;
  • в списке Отчеты по управлению персоналом подсистемы Наряды и работы.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по выводимым данным в отчете можно отредактировать параметры:

  • Период отчета - отбор трудовых затрат за указанный период;
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года;
  • По регламентным мероприятиям - настройка, позволяющая учитывать трудовые затраты ремонта из документов  "График регламентных мероприятий";
  • По план-графикам - настройка, позволяющая учитывать трудовые затраты ремонта из документов "План-график ППР";
  • По внешним основаниям - настройка, позволяющая учитывать трудовые затраты ремонта из документов "Внешнее основания для работ";
  • По выявленным дефектам - настройка, позволяющая учитывать трудовые затраты ремонта из документов "Выявленные дефекты".

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует два предопределенных варианта отчета:

  1. Анализ использования персонала: по объектам ремонта - открывается по умолчанию. Данные о сотрудниках сгруппированы по объектам ремонта, по которым выполнялись ремонты за указанный период. Фиксация планового времени работы сотрудников осуществляется документом "Прием на работу", фактического -  документами "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия":
    • Объект ремонта - наименование объекта ремонта;
    • Вид ремонта - вид ремонта (поле "Вид ремонта" из документа "План-график ППР"; поле "Вид мероприятия" из документа "График регламентных мероприятий"; поля "Вид ремонта при вводе на основании выявленных дефектов" и "Вид ремонта при вводе на основании внешних оснований" из настроек параметров подсистемы ТОИР);
    • Сотрудник - ФИО сотрудника организации пользователя;
    • Период - периодичность отчета для вывода данных в разрезе дня/недели/месяца/квартала/года (зависит от параметра отчета "Периодичность");
    • Время работы - время работы сотрудника в зависимости от объекта ремонта и вида ремонта:
      • Факт - фактическое время работы сотрудника фиксируется документами "Акт о выполнении этапа работ" (поле "Время работы" на вкладке "Исполнители - По операциям"), "Акт о выполнении регламентного мероприятия" (поле "Время работы" на вкладке "Исполнители - Хозспособ");
      • План - плановое время работы сотрудника фиксируется документами "Прием на работу" (поле "График работы") или "Кадровый перевод" (при изменении графика работы (ФО "Перевести на другой график"));
    • Итоговое время работы - итоговое время работы сотрудника, которое потребовалось для выполнения ремонтной работы в рамках объекта ремонта за указанный период.
  2. Анализ использования персонала: по сотрудникам - вариант отчета, в котором данные по объектам ремонта и видам ремонта сгруппированы по сотрудникам.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
