Отчет «КТГ оборудования»

Отчет предназначен для получения информации о времени нахождения оборудования в эксплуатации, простое, на испытаниях относительно графику работы оборудования. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет состояния оборудования.

Открывается по команде КТГ оборудования: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - период построения отчета;
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года;
  • Структура иерархии - по умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").
Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианты отчета:

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианты отчета:

  1. КТГ оборудования: таблица - открывается по умолчанию. Данные об отказах сгруппированы по организации/подразделению. В этом варианте отчета по умолчанию в строках выводится информация об объекте ремонта, длительности времени нахождения в состоянии, фонде времени и коэффициенте готовности оборудования:
    • Организация - наименование организации, к которой относится объект ремонта (поле "Организация" из карточки объекта ремонта); 
    • Подразделение - наименование подразделения организации, к которому относится объект ремонта (поле "Подразделение" из карточки объекта ремонта);
    • Объект ремонта - наименование объекта ремонта;
    • Эксплуатация, простой, испытания - время нахождения в состоянии с соответствующим типом в заданном периоде времени;
    • Фонд времени - рабочее время по графику работы оборудования в заданном периоде времени;
    • КТГ,% - коэффициент технической готовности оборудования. Рассчитывается по формуле: (Эксплуатация/Фонд времени) *100%.
  2. КТГ оборудования: диаграмма- отчет предназначен для анализа времени нахождения оборудования в состоянии (эксплуатация, простой, испытания) в графическом виде.

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
