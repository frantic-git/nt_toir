Отчет «Ведомость по товарам на складах»

Отчет предназначен для получения информации о количественном движении товаров на складах предприятия за период.
Открывается по команде Ведомость по товарам на складах: в списке Отчеты подсистемы МТО.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду перемещения товаров на складах предприятия в отчете можно отредактировать параметр отчета: Период.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Ведомость по товарам на складах. Данные о перемещениях товаров сгруппированы по складам.
В этом варианте отчета по умолчанию в строках выводится информация о товарах (их артикулах, наименованиях, единицах измерения, характеристиках - если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры) и их количествах (начальный и конечный остаток, приход и расход) на складах предприятия. Приход товаров на склад выполняется документами "Поступление товаров и услуг", "Перемещение товаров" и "Оприходования излишков товаров" , а списание товаров со склада - документами "Внутренние потребления товаров", "Списания недостач товаров" и "Перемещение товаров":

  • Склад - склад предприятия, на котором хранятся товары (поле "Склад" из документов "Поступление товаров и услуг", "Перемещение товаров", "Оприходования излишков товаров", "Списания недостач товаров" и "Внутренние потребления товаров");
  • Артикул - артикул товара, указанный в его карточке (поле "Артикул" из справочника "Номенклатура");
  • Номенклатура, Ед. изм. - наименование товара и его единица измерения (поля "Рабочее наименование" и "Единица хранения" из справочника "Номенклатура");
  • Характеристика - наименование характеристики товара (поле "Характеристика" из документов "Поступление товаров и услуг", "Перемещение товаров", "Оприходования излишков товаров", "Списания недостач товаров" и "Внутренние потребления товаров"). Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
  • Начальный остаток - количество товаров, находящееся на складе на начало указанного периода;
  • Приход - количество товаров, поступившее на склад за указанный период отчета;
  • Расход - количество товаров, списанное со склада за указанный период отчета;
  • Конечный остаток - количество товаров, оставшееся на складе к концу указанного периода (рассчитывается по формуле: Начальный остаток + Приход - Расход).
Под основной таблицей находится обобщенная таблица, в которой выводится информации о количественном движении товаров на складах предприятия за период в разрезе единиц измерения.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
