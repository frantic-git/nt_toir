Отчет «Статистика потребления с детализацией по месяцам»

Отчет предназначен для получения информации о статистике потребления номенклатуры за указанный период. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов – МТО: Учитывать статистику потребления. 

Открывается по команде Статистика потребления номенклатуры с детализацией по месяцам: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значения параметра отчета: Период (период отчета не должен выходить за переделы одного года).

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:

  1. Статистика потребления с детализацией по месяцам - открывается по умолчанию. Данные о потребление номенклатуры группированы по складам. Фиксация потребления осуществляется документами "Внутреннее потребление товаров":
    • Склад - наименование склада;
    • Номенклатура - наименование номенклатуры;
    • Артикул - артикул номенклатуры;
    • Серия - серия номенклатуры. Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать серии номенклатуры;
    • Характеристика - наименование характеристики номенклатуры. Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
    • Статистика - статистика потребления. Рассчитывается по формуле = Потребление за месяц/ Количество дней в месяце:
      • Максимум - максимальная статистика потребления;
      • Минимум - минимальная статистика потребления;
      • Среднее - средняя статистика потребления.
    • Потребление - потребление номенклатуры, выполненное документами "Внутреннее потребление товаров":
      • Максимум - максимальное потребление за аналогичные периоды*;
      • Минимум - минимальное потребление за аналогичные периоды*;
      • Среднее - среднее потребление за аналогичные периоды*.
  2. Стоимость потребления номенклатуры с детализацией по месяцам - вариант отчета, в котором выводится информация о стоимости потребления номенклатуры за указанный период. Стоимость определяется согласно настройке "Тип цены для расчеты фактической себестоимости ремонтов" в разделе Настройка параметров системы - Интерфейс и ввод документов - Общие:

    • Потребление - стоимость потребление номенклатуры, зафиксированная документом  "Установка цен номенклатуры":
      • Максимум - максимальное потребление за аналогичные периоды*;
      • Минимум - минимальное потребление за аналогичные периоды*;
      • Среднее - среднее потребление за аналогичные периоды*.
* Количество лет, за которое выполняется расчет, берется из настройки "Период расчета статистик потребления". Настройка находится в разделе Настройка параметров системы - Интерфейс и ввод документов – МТО: Период расчета статистики потребления.

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
