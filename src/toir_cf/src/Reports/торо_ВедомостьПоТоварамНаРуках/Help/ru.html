Отчет «Ведомость по товарам на руках»

Отчет предназначен для получения информации о количественном движении товаров, выданных под заказы на внутреннее потребление.
Открывается по команде Ведомость по товарам на руках: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду перемещения товаров на складах предприятия в отчете можно отредактировать параметр отчета: Период.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Ведомость по товарам на руках. Данные о перемещениях товаров сгруппированы по заказам на внутреннее потребление.
В этом варианте отчета по умолчанию в строках выводится информация о товарах (их артикулах, наименованиях, единицах измерения, характеристиках - если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры) и их количествах (начальный и конечный остаток, приход и расход) на складах предприятия. Приход товаров на руки выполняется документом "Передача товаров на руки", а списание - документом "Возврат товаров на склад".

  • Заказ на внутреннее потребление - заказ, под который выданы товары;
  • Артикул - артикул товара, указанный в его карточке (поле "Артикул" из справочника "Номенклатура");
  • Номенклатура, Ед. изм. - наименование товара и его единица измерения (поля "Рабочее наименование" и "Единица хранения" из справочника "Номенклатура");
  • Характеристика - наименование характеристики товара (поле "Характеристика" из документов "Поступление товаров и услуг", "Перемещение товаров", "Оприходования излишков товаров", "Списания недостач товаров" и "Внутренние потребления товаров"). Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
  • Начальный остаток - количество товаров, выданное под заказы на внутреннее потребление на начало указанного периода;
  • Приход - количество товаров, выданное под заказы на внутреннее потребление за указанный период отчета;
  • Расход - количество товаров, возвращенное на склад за указанный период отчета;
  • Конечный остаток - количество товаров, оставшееся на руках к концу указанного периода (рассчитывается по формуле: Начальный остаток + Приход - Расход).
Под основной таблицей находится обобщенная таблица, в которой выводится информации о количественном движении товаров на складах предприятия за период в разрезе единиц измерения.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
