Отчет «Фактические затраты по объектам ремонта»

Отчет предназначен для проведения анализа фактических затрат по объектам ремонта. Данные о затратах представляются согласно номенклатуре для каждого объекта ремонта в разрезе вида ремонта, вида затрат.
Открывается по команде Фактические затраты по объектам ремонта в списке Отчеты по оборудованию и нормативам подсистемы Оборудование и нормативы.

Параметры отчета 
Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:
  • Период отчета - отбор фактических затрат по плановым датам;
  • По дефектам - настройка, позволяющая учитывать затраты по документам "Выявленные дефекты";
  • По ППР - настройка, позволяющая учитывать затраты по документам "План-график ППР";
  • По предписаниям - настройка, позволяющая учитывать затраты по документам "Внешнее основания для работ";
  • По регламентным мероприятиям - настройка, позволяющая учитывать затраты по документам  "График регламентных мероприятий";
  • Выполняемые подрядчиками - настройка, позволяющая учитывать затраты по ремонтам, выполняемым подрядчиками;
  • Затраты по запчастям - настройка, позволяющая учитывать затраты по запчастям;
  • Материальные затраты - настройка, позволяющая учитывать материальные затраты;
  • Трудозатраты - настройка, позволяющая учитывать трудозатраты;
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии");
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года.
Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны три предопределенных варианта отчета:

  1. Фактические затраты по объектам ремонта: таблица - открывается по умолчанию. Данные о фактических затратах сгруппированы по подразделению/контрагенту.
    В этом варианте отчета по умолчанию в строках выводится информация о фактических затратах (материальных, трудовых, затратах запчастей) в количественном и стоимостном выражении. Фиксация фактических затрат выполняется документами "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия":
    • Подразделение/Контрагент - наименование подразделения/контрагента, указанного у объекта ремонта в качестве исполнителя ремонта (Поле "Исполнитель" из документа "Акт о выполнении этапа работа" на закладке "Исполнители", поле "Контрагент" из документа "Акт о выполнении регламентного мероприятия" на закладке "Исполнители");
    • Объект ремонта - наименование объекта ремонта;
    • Ремонтная работа - наименование технологической операции, для выполнения которой требуются материальные затраты (поле "Ремонтная работа" из дерева ремонтных работ документов "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия");
    • Вид затрат - общие затраты, разделенные на три группы: материальные, трудовые, запчасти;
    • Исполнитель/Номенклатура -  наименование номенклатуры или исполнителя, требуемых для выполнения ремонтной работы (поле "Номенклатура" или "Сотрудник" из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Материальные затраты" или "Исполнители");
    • Единица измерения - наименование единицы измерения номенклатуры, требуемой для выполнения ремонтной работы (поле "Ед. изм." из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Материальные затраты");
    • Характеристика - наименование характеристики номенклатуры (поле "Характеристика" из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия"). Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
    • Квалификация - наименование квалификации ремонтного персонала (поле "Квалификация" из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Исполнители");
    • Количество (в разрезе месяца) - количество, которое фактически затрачено на выполнение ремонтной работы (поле "Количество" из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Материальные затраты");
    • Сумма затрат по объектам ремонта (в разрезе месяца) - сумма фактических затрат на материальные и трудовые ресурсы, запчасти, которая потребовалась для выполнения ремонтной работы (рассчитывается по формуле: Фактическое количество * Цена на фактическую дату ремонта. Цена номенклатуры устанавливается документом "Установка цен номенклатуры". Для трудовых затрат - поле "Стоимость работ" из документов "Акт о выполнении этапа работ", "Акт о выполнении регламентного мероприятия" на закладке "Исполнители");
    • Итоговое количество - итоговое количество, которое фактически затрачено на выполнение ремонтной работы за указанный период;
    • Итоговая сумма затрат по объектам ремонта - итоговая сумма фактических затрат на материальные и трудовые ресурсы, запчасти, которая потребовалась для выполнения ремонтной работы за указанный период.
  2. Фактические затраты по объектам ремонта: таблица (с иерархией) - вариант отчета, в котором фактические затраты ремонтов сгруппированы по объекту иерархии, объекту ремонта:
    • Объект иерархии - наименования всех родителей объекта ремонта (объекты ремонта или группы объектов ремонта).
  3. Фактические затраты по объектам ремонта: диаграмма - вариант отчета, в котором фактические затраты отображаются в разрезе объектов ремонта в графическом виде по всем ремонтным работам за выбранный период.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
