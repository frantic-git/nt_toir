Отчет «Рецидивные дефекты»

Отчет предназначен для получения информации о рецидивных дефектах и их количестве за период.

Рецидивный дефект определяется точным совпадением параметров с одним из предыдущих дефектов (у которых дата обнаружения меньше, чем у текущего):

  • Объект ремонта + Отказавший элемент + Типовой дефект (если типовой дефект заполнен);
  • Объект ремонта + Отказавший элемент + Вид дефекта + Причина дефекта (если типовой дефект не заполнен).


Открывается по команде Рецидивные дефекты: в списке Отчеты по показателям эксплуатации в подсистеме Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - фильтр по дате обнаружения текущих дефектов;
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Рецидивные дефекты. Данные сгруппированы по объектам ремонта и отказавшим элементам.

Основные поля отчета:

  • Объект ремонта и отказавший элемент - объект ремонта и отказавший элемент из документа выявленных дефектов;
  • Вид ремонта и дата обнаружения - вид ремонта и дата обнаружения рецидивного и предыдущего дефектов;
  • Типовой дефект, вид и причина дефекта - значения из документа выявленного дефекта по рецидивному ремонту;
  • Количество - количество рецидивных дефектов оборудования. Данное значение выводится только на вернем уровне и вычисляется путем подсчета рецидивных.

При расшифровке ссылочных полей (объект ремонта, типовой дефект и т.д.) открываются соответствующие значения, при расшифровке остальных полей - выявленный дефект рецидивного или предыдущего дефекта.

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
