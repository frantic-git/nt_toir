Отчет «Остатки номенклатуры на руках»

Отчет предназначен для получения информации о количественном и стоимостном учете номенклатуры, выданной на руки под заказы на внутреннее потребление на выбранную дату. 
Открывается по команде Остатки номенклатуры на руках: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по наличию номенклатуры на складах предприятия на конкретную дату в отчете можно отредактировать параметр отчета: На дату.

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Остатки номенклатуры на руках. Данные о наличии номенклатуры на складах сгруппированы по заказам на внутреннее потребление. 
В этом варианте отчета по умолчанию в строках выводится информация о номенклатуре (ее наименовании, единице измерения, характеристике - если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры) и ее количестве, цене и стоимости. Остаток номенклатуры на складах регулируется документам "Передача товаров на руки" и "Возврат товаров на склад".

  • Заказ на внутреннее потребление - заказ, под который выданы товары;
  • Номенклатура, Единица хранения - наименование номенклатуры и его единица хранения (поля "Рабочее наименование" и "Единица хранения" из справочника "Номенклатура");
  • Характеристика - наименование характеристики номенклатуры (поле "Характеристика" из документов "Поступление товаров и услуг", "Перемещение товаров", "Оприходования излишков товаров", "Списания недостач товаров" и "Внутренние потребления товаров"). Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
  • Количество - количество номенклатуры (остаток номенклатуры), находящееся на руках на указанную дату;
  • Цена - цена номенклатуры, хранящейся на складе на указанную дату. Устанавливается документом "Установка цен номенклатуры";
  • Стоимость - стоимость номенклатуры, хранящейся на складе на указанную дату (рассчитывается по формуле: Количество * Цена).

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
