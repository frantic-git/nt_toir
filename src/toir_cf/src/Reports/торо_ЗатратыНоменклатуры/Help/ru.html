Отчет «Затраты номенклатуры»

Отчет предназначен для план-фактного анализа материальных затрат по ремонтным работам за период.
Открывается по команде Затраты номенклатуры: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - отбор материальных затрат по плановым датам ремонтов;
  • Заявка на ремонт - отбор материальных затрат по конкретному документу "Смета ремонта (Заявка на ремонт)".

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:

  1. Затраты номенклатуры: таблица - открывается по умолчанию. Данные о материальных затратах сгруппированы по технологическим операциям.
    В этом варианте отчета по умолчанию в строках выводится информация о плановых и фактических материальных затратах (а так же отклонение плановых затрат от фактических) в количественном и стоимостном выражении. Фиксация плановых затрат выполняется документом "Смета ремонта (Заявка на ремонт)", фактических затрат - документом "Акт о выполнении этапа работ":
    • Ремонтная работа - наименование технологической операции, для выполнения которой требуются материальные затраты (поле "Ремонтная работа" из дерева ремонтных работ документов "Смета ремонта (Заявка на ремонт)" и "Акт о выполнении этапа работ");
    • Номенклатура - наименование номенклатуры, требуемой для выполнения ремонтной работы (поле "Номенклатура" из документов "Смета ремонта (Заявка на ремонт)" и "Акт о выполнении этапа работ" на закладке "Материальные затраты");
    • Единица измерения - наименование единицы измерения номенклатуры, требуемой для выполнения ремонтной работы (поле "Ед. изм." из документов "Смета ремонта (Заявка на ремонт)" и "Акт о выполнении этапа работ" на закладке "Материальные затраты");
    • Плановое количество - количество, которое запланировано к выполнению ремонтной работы (поле "Количество" из документов "Смета ремонта (Заявка на ремонт)" на закладке "Материальные затраты");
    • Плановая сумма - стоимость плановых материальных затрат, необходимых для выполнения ремонтной работы (рассчитывается по формуле: Плановое количество * Цена на плановую дату ремонта. Цена номенклатуры устанавливается документом "Установка цен номенклатуры");
    • Фактическое количество - количество, которое фактически затрачено на выполнение ремонтной работы (поле "Количество" из документов "Акт о выполнении этапа работ" на закладке "Материальные затраты");
    • Фактическая сумма - стоимость фактических материальных затрат, требуемых для выполнения ремонтной работы (рассчитывается по формуле: Фактическое количество * Цена на фактическую дату ремонта);
    • Отклонение количество - отклонение планового количества материальных затрат от фактического (рассчитывается по формуле: Плановое количество - Фактическое количество);
    • Отклонение сумма - отклонение плановой суммы материальных затрат от фактической (рассчитывается по формуле: Плановая сумма - Фактическая сумма).
  2. Затраты номенклатуры: диаграмма - отчет предназначен для анализа отклонений суммы (фактической от плановой) в разрезе номенклатуры в графическом виде по всем ремонтным работам за выбранный период.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
