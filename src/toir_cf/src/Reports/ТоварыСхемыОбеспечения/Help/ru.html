Отчет «Товары, включенные в схему обеспечения»

Отчет предназначен для анализа способов обеспечения товаров по складам в рамках выбранной схемы обеспечения. Служит расшифровкой схемы обеспечения, детализируя информацию о номенклатуре, которая обеспечивается в рамках схемы. 

Открывается по гиперссылке Всего рядом с полем Используется для товаров в форме элемента справочника "Схемы обеспечения".

Варианты отчета


С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступен и открывается по умолчанию один предопределенный вариант отчета – Товары, включенные в схему обеспечения. Данные по номенклатуре группируются по виду номенклатуры в разрезе складов и способов обеспечения потребностей и детализируются до артикула и наименования номенклатуры:

  • Вид номенклатуры – наименование вида номенклатуры.
  • Артикул – артикул номенклатуры.
  • Номенклатура – наименование номенклатуры.
  • Склад – наименования складов из выбранной схемы обеспечения.
  • Способ обеспечения потребностей – наименование способа обеспечения номенклатуры на складе в рамках выбранной схемы обеспечения.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
