Отчет «Текущее состояние оборудования»

Отчет предназначен для просмотра текущего состояния оборудования. Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет состояния оборудования.
Открывается по команде Текущее состояние оборудования: в списке Отчеты по оборудованию и нормативам подсистемы Оборудование и нормативы.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета: 

  • Дата отчета - отбор по дате (выводится текущее состояние оборудования на эту дату);
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный варианта отчета - Текущее состояние оборудования. Данные о текущем состоянии оборудования сгруппированы по объектам ремонта. Отчет формируется за указанную дату. Регистрация состояний объектов ремонта выполняется документами "Состояния объектов ремонта" и "Ввод начальных данных":

  • Объект иерархии - объект ремонта или группа объектов ремонта, имеющие подчиненные объекты, т.е. родитель объекта ремонта;
  • Объект ремонта - наименование объекта ремонта;
  • Дата изменения состояния - дата и время изменения состояния объекта ремонта (поля "Дата изменения состояния", "Начало периода", "Окончание периода" из документа "Состояния объектов ремонта"; поле "от" из документа "Ввод начальных данных");
  • Вид эксплуатации - вид эксплуатации объекта ремонта с даты изменения состояния (поля "Вид эксплуатации", "Вид эксплуатации на период" из документа "Состояния объектов ремонта"; поле "Начальное состояние" из документа "Ввод начальных данных"). 
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
