Отчет «График работы сотрудника»

Отчет выводит данные в виде печатной формы. Он предназначен для вывода графика работы сотрудника с учетом введенных документов Плановая доступность персонала.

Открывается по команде График работы сотрудника: из документа Плановая доступность персонала и обработки Управление плановой доступностью персонала подсистемы Управление персоналом.

Параметры отчета 

Параметры указываются в форме настроек вывода графика работы при открытии отчета. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Год - период построения графика работы;
  • Сотрудники - список сотрудников, для которых формируются графики работы;
