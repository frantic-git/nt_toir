Отчет «Сводная диаграмма графиков ППР»

Отчет предназначен для получения информации о ремонтных работах, запланированных во всех план-графиках ППР за указанный период в графическом виде. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Планирование ТОиР: Использовать ППР.
Открывается по команде Сводная диаграмма графиков ППР: в списке Отчеты по планированию ТОиР подсистемы Планирование ТОиР.

Параметры отчета

Для фильтра по выводимым данным в отчете можно отредактировать параметры отчета*:

  • Объект ремонта - отбор по объектам ремонта;
  • Вид ремонта - отбор по видам ремонта;
  • Организация/ контрагент - отбор по организации (или контрагенту), к которой принадлежат объекты ремонта;
  • Отображать иерархию - отбор, позволяющий вывести родителей объектов ремонта в зависимости от выбранной иерархии;
  • Период с ... по - отбор по периоду планирования ремонтных работ;
  • Подразделение-владелец - отбор по подразделению-владельцу, к которому принадлежат объекты ремонта;
  • Подразделение-исполнитель - отбор по подразделению-исполнителю, которое выбрано в качестве исполнителя ремонтных работ.

Варианты отчета

Существует один варианта отчета - Сводная диаграмма графиков ППР. Данные о планово-предупредительных ремонтах представлены в виде диаграммы Ганта. Все ремонты в диаграмме сгруппированы по объектам ремонта.
Планирование ремонтных работ выполняется документами "План-график ППР" (даты ремонтов могут быть изменены корректировкой ППР, обработкой "Закрытие заявок и ремонтов", документом "Остановочный ремонт" или обработкой "Общий план работ").

Примечания*:
Существуют дополнительные механизмы, позволяющие расширить возможности отчета:
  1. Отображать замещенные - позволяет выводить замещенные ремонты в диаграмму;
  2. Сохранить как изображение - позволяет сохранять сформированную диаграмму на компьютер в формате изображений (по умолчанию .png);
  3. Печать - позволяет распечатать диаграмму;
  4. Сортировать ОР - позволяет сортировать ОР по алфавиту;
  5. Развернуть/свернуть все - позволяет разворачивать/ сворачивать дерево диаграммы, содержащее объекты ремонта (и родители объектов, если включена настройка "Отображать иерархию") и виды ремонта;
  6. Ползунок - позволяет масштабировать отображение ремонтов по дням/ неделям/ месяцам/ годам.
