Отчет «Статистика потребления с детализацией по годам»

Отчет предназначен для получения информации о статистике потребления номенклатуры за указанный период. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов – Управление МТО: Учитывать статистику потребления. 

Открывается по команде Статистика потребления номенклатуры с детализацией по месяцам: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значения параметра отчета: Период.

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:

  1. Статистика потребления с детализацией по годам - открывается по умолчанию. Данные о потребление номенклатуры группированы по складам. Фиксация потребления осуществляется документами "Внутреннее потребление товаров":
    • Склад - наименование склада;
    • Артикул - артикул номенклатуры;
    • Номенклатура - наименование номенклатуры;
    • Характеристика - наименование характеристики номенклатуры. Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры;
    • Серия - серия номенклатуры. Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать серии номенклатуры;
    • Статистика - статистика потребления. Рассчитывается по формуле = Потребление за год/ Количество дней в году:
      • Общее - суммарное потребление за год/ количество дней в году;
      • Страховое - общая статистика потребления за год с учетом страхового запаса.
    • Потребление - потребление номенклатуры, выполненное документами "Внутреннее потребление товаров":
      • Общее - суммарное потребление за год;
      • Страховое - потребление номенклатуры за год с учетом страхового запаса.
  2. Стоимость потребления номенклатуры с детализацией по годам - вариант отчета, в котором выводится информация о стоимости потребления номенклатуры за указанный период. Стоимость определяется согласно настройке "Тип цены для расчеты фактической себестоимости ремонтов" в разделе Настройка параметров системы - Интерфейс и ввод документов - Общие:

    • Потребление - стоимость потребления номенклатуры, зафиксированная документом "Установка цен номенклатуры":
      • Стоимость - суммарная стоимость потребления за год;
      • Стоимость (страховое) - стоимость потребления номенклатуры за год с учетом страхового запаса.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
