Отчет «Длительность согласования документов»

Отчет предназначен для получения информации о длительности согласования документов за период. 
Открывается по команде Длительность согласования документов в подсистеме Мониторинг показателей процессов. В отчете выбираются только те документы, для которых включено согласование (при условии включенного согласования соответствующего блока документов: Настройка параметров системы - Согласование: Ремонты, Регламентные мероприятия, МТО).

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - фильтр по дате согласуемого документа;
  • Выводить документы по ремонтам - настройка, позволяющая учитывать данные по документам, указанным в блоке согласования ремонтов (Настройка параметров системы - Согласование: Ремонты);
  • Выводить документы по регламентным мероприятиям - настройка, позволяющая учитывать данные по документам, указанным в блоке согласования регламентных документов (Настройка параметров системы - Согласование: Регламентные мероприятия). При отключенном флаге регламентных мероприятий (Настройка параметров системы - Интерфейс и ввод документов - Планирование ТОИР: Использовать регламентные мероприятия) настройка скрывается с формы настроек отчета (обычный режим);
  • Выводить документы по МТО - настройка, позволяющая учитывать данные по документам, указанным в блоке согласования МТО (Настройка параметров системы - Согласование: МТО).

Варианты отчета 

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Длительность согласования документов. Данные о согласовании сгруппированы по организациям и подразделениям. 
В этом варианте отчета по умолчанию в строках выводится информация о согласовании конкретного документа.

Основные поля отчета:

  • Организация и Подразделение - реквизиты шапки документа;
  • Документ - ссылка на согласуемый документ;
  • Статус - текущий статус согласования документа;
  • Согласование завершено - флаг, показывающий, что документ находится в финальном статусе согласования;
  • Дата начала согласования - дата установки первого статуса согласования по этому документу (дата первой записи из истории согласования);
  • Дата окончания согласования - дата установки финального статуса согласования документа (дата последней записи из истории согласования). Если согласование не завершено, не заполняется;
  • Время согласования - время (часы, минуты, секунды) между датами начала и окончания согласования. Рассчитывается только если согласование завершено, в противном случае пусто. Показывается на уровне детальных записей в одной колонке. На уровне группировок и итоговых записей выводятся две колонки - среднее и максимальное значение. При расчете среднего значения не учитывает документы, согласование для которых не завершено (т.е. их наличие не должно уменьшать среднее значение).

Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
