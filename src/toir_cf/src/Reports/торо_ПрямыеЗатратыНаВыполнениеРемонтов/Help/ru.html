Отчет «Прямые затраты на выполнение ремонтов»

Отчет содержит информацию о плановых затратах на трудовые ресурсы и МТО, необходимые для выполнения запланированных ТО и ремонтов на определенный период.

Открывается по команде Прямые затраты на выполнение ремонтов: в списке Отчеты по планированию ТОиР подсистемы Планирование ТОиР.

Информация в отчете группируется для подрядного и хозспособа выполнения ремонтов и представлена в разрезе объектов ремонта и их нормативных ТО и ремонтов. Предусмотрена возможность включения в затраты только ТО и ремонтов для объектов ремонта с большей критичностью. 

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:
  • Период отчета — отбор ремонтов по плановым датам;
  • Расчет трудозатрат — отбор ремонтов по способу расчета трудозатрат: по расценкам тех. операций или по стоимости нормочаса квалификации.
  • Способ выполнения — отбор ремонтов по способу выполнения: хозспособу или подрядному.
  • Учитывать ППР — настройка, позволяющая учитывать ремонты из документов "План-график ППР";
  • Учитывать регламентные мероприятия — настройка, позволяющая учитывать ремонты из документов  "График регламентных мероприятий";
  • Учитывать Сметы ремонта (Заявки на ремонт) по дефектам — настройка, позволяющая учитывать ремонты из документов "Выявленные дефекты";
  • Учитывать Сметы ремонта (Заявки на ремонт) по внешним основаниям — настройка, позволяющая учитывать ремонты из документов "Внешнее основания для работ";
  • Критичность — отбор ремонтов по параметру критичности объектов ремонта.
  • Структура иерархии — отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианта отчета:
  1. Прямые затраты на выполнение ремонтов: таблица — открывается по умолчанию. В этом варианте отчета прямые затраты выводятся в виде таблицы. Данные группируются по видам затрат и детализируются до критичности и наименования объекта ремонта, способа выполнения и вида ремонта, а также его даты. Также выводятся итоговые суммы по каждому из разрезов затрат и итоговые суммы по отдельным ремонтам.
    • Критичность — критичность объекта ремонта.
    • Способ выполнения — способ выполнения ремонта: хозспособ или подрядный.
    • Объект ремонта — наименование объекта ремонта;
    • Вид ремонта — вид ремонта (поле "Вид ремонта" из документа "План-график ППР"; поле "Вид мероприятия" из документа "График регламентных мероприятий"; поля "Вид ремонта при вводе на основании выявленных дефектов" и "Вид ремонта при вводе на основании внешних оснований" из настроек параметров подсистемы ТОИР);
    • Дата ремонта — дата ремонта (поле "Период" из документов "План-график ППР" и "График регламентных мероприятий", поле "Дата начала" из документа "Смета ремонта (Заявка на ремонт)").
    • Трудозатраты сумма — плановая стоимость всех трудозатрат на ремонт.
    • Материальные затраты сумма — плановая стоимость всех материальных затрат на ремонт.
    • Затраты запчастей сумма —  плановая стоимость всех затрат на запчасти.
    • Всего сумма — общая стоимость всех затрат на ремонт.
  2. Прямые затраты на выполнение ремонтов: диаграмма — вариант отчета, в котором прямые затраты на выполнение ремонтов отображаются в разрезе объектов и видов ремонта в графическом виде за выбранный период.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
