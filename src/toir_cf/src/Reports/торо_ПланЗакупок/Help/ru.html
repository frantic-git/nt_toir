Отчет «План закупок»

Отчет предназначен для получения информации о датах плановых поступлений номенклатуры в разрезе организации и склада.
Открывается по команде План закупок: в списке Отчеты по МТО подсистемы МТО.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по дате поставки в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует два предопределенных варианта отчета:
  1. План закупок - открывается по умолчанию. Данные группируются по организации и складу. Фиксация плановых закупок фиксируются документом "Заказ поставщику".
    • Организация - наименование организации, в которой оформлен документ "Заказ поставщику";
    • Склад - наименование склада поступления номенклатуры;
    • Номенклатура - наименование номенклатуры;
    • Характеристика - наименование характеристики номенклатуры (выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры);
    • Единица измерения - наименование единицы измерения номенклатуры;
    • Контрагент - наименование контрагента, у которого осуществляется заказ номенклатуры;
    • Заказ поставщику - ссылка на документ "Заказ поставщику";
    • Дата поставки - дата поставки из документа "Заказ поставщику";
    • Количество заказанное - заказанное количество номенклатуры.
  2. План поставок под заказы на внутренние потребления- вариант отчет, предназначенный для отслеживания плана поставок под заказы на внутренние потребления. Потребность отражается документом "Заказ на внутреннее потребление", поставки - документом "Заказ поставщику":
    • Организация - наименование организации, в которой оформлен документ "Заказ поставщику";
    • Дата потребности - поле "Дата отгрузки" из документа "Заказ на внутреннее потребление";
    • Номенклатура - наименование номенклатуры;
    • Заказ на ВП - ссылка на документ "Заказ на внутреннее потребление";
    • Характеристика - наименование характеристики номенклатуры (выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры);
    • Объект ремонта - наименование объекта ремонта, для которого необходимо обеспечить потребность;
    • Единица измерения - наименование единицы измерения номенклатуры;
    • Вид ремонта - наименование вида ремонта, для которого необходимо обеспечить потребность;
    • Заказ поставщику - ссылка на документ "Заказ поставщику";
    • Количество требуемое - требуемое количество номенклатуры для обеспечения ремонтов;
    • Количество закупленное - закупленное количество номенклатуры;
    • Дата поставки - дата поставки из документа "Заказ поставщику".
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
