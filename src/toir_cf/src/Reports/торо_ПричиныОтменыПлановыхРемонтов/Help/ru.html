Отчет «Причины отмены плановых ремонтов»

Отчет предназначен для получения информации о датах и причинах отмены или переноса плановых ремонтов за период. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Планирование ТОиР: Использовать ППР.
Открывается по команде Причины отмены плановых ремонтов: в списке Отчеты по планированию ТОиР подсистемы Планирование ТОиР.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду плановых ремонтов в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один вариант отчета - Причины отмены плановых ремонтов.
Данные об отмененных или перенесенных плановых ремонтах сгруппированы по категориям ремонтов (отмененные/перенесенные), подразделениям/контрагентам объектов ремонта и объектам ремонта.
В этом варианте отчета по умолчанию в строках выводится информация о видах ремонта, плановых датах начала и окончания до и после корректировки ремонтов и соответствующих им причинах закрытия. Планирование ремонтов объектов ремонта отражается документом "План-график ППР", а регистрация отмены или переноса планового ремонта - обработкой "Закрытие заявок и ремонтов" (из нее автоматически формируется документ "Закрытие заявок и ремонтов"):

  • Перенесен/отменен - категория ремонта (флаг "Перенесен"/ "Отменен" из документа "Закрытие заявок и ремонтов");
  • Подразделение/ Контрагент - подразделение объекта ремонта (поле "Подразделение" из карточки объекта ремонта)/ контрагент, к которому относится объект ремонта (поле "Контрагент" из карточки объекта ремонта);
  • Объект ремонта - наименование объекта ремонта;
  • Вид ремонта - вид запланированного ремонта из документа "План-график ППР";
  • Дата начала до корректировки - дата начала запланированного ремонта из документа "План-график ППР";
  • Дата окончания до корректировки - дата окончания запланированного ремонта из документа "План-график ППР";
  • Дата начала - дата начала перенесенного ремонта из документа "Закрытие заявок и ремонтов" (поле "Новая дата начала" из обработки/документа "Закрытие заявок и ремонтов"), для отмененного ремонта новая дата начала не заполняется;
  • Дата окончания - дата окончания перенесенного ремонта из документа "Закрытие заявок и ремонтов" (поле "Новая дата окончания" из обработки/документа "Закрытие заявок и ремонтов"), для отмененного ремонта новая дата окончания не заполняется;
  • Причина закрытия - причина отмены или переноса планового ремонта из документа "Закрытие заявок и ремонтов" (поле "Причина закрытия ремонта" из обработки "Закрытие заявок и ремонтов" - в документе "Закрытие заявок и ремонтов" поле "Причина закрытия");
  • Ответственный - ответственный пользователь, который зарегистрировал документ "Закрытие заявок и ремонтов" (поле "Ответственный" в документе "Закрытие заявок и ремонтов").
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
