Отчет «Анализ отказов оборудования»

Отчёт используется для получения информации об отказах оборудования за период. В отчете для каждого объекта ремонта выводятся причина и количество раз отказа оборудования. Число отказов совпадает с числом документов Выявленный дефект, созданных для объекта ремонта. Доступен при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов - Показатели эксплуатации: Учет выявленных дефектов оборудования.

Открывается по команде Анализ отказов оборудования: в списке Отчеты по показателям эксплуатации подсистемы Показатели эксплуатации.

Параметры отчета 

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - отбор отказов по дате их регистрации в документе "Выявленные дефекты";
  • Периодичность - отбор, позволяющий отобразить период отчета в разрезе дня/недели/месяца/квартала/года;
  • Структура иерархии - по умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии").
Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступны два предопределенных варианты отчета:

  1. Анализ отказов оборудования: таблица - открывается по умолчанию. Данные об отказах сгруппированы по организации/подразделению.

В этом варианте отчета по умолчанию в строках выводится информация об отказавшем элементе и причине его дефекта. Так же, выводится информация о количестве отказов, которая фиксируется документом "Выявленные дефекты". Длительность ремонта, фактические материальные и трудовые затраты фиксируются документом "Акт о выполнении этапа работ": 

    • Организация/Контрагент - организация объекта ремонта (поле "Организация" из карточки объекта ремонта)/ контрагент, к которому относится объект ремонта (поле "Контрагент" из карточки объекта ремонта);
    • Объект ремонта - наименование объекта ремонта;
    • Отказавший элемент - поле "Отказавший элемент" из документа "Выявленные дефекты";
    • Причина дефекта - поле "Причина дефекта" из документа "Выявленные дефекты";
    • Количество - количество введенных документов "Выявленные дефекты" на указанный период; 
    • Длительность ремонта - количество времени, затраченное на устранение отказа (разница между полями "Дата начала" и "Дата окончания" документа "Акт о выполнении этапа работ" (с завершающим флагом));
    • Фактические материальные затраты - стоимость фактических затраты на материальные ресурсы (рассчитывается по формуле: Фактическое количество * Цена на фактическую дату ремонта. Цена номенклатуры устанавливается документом "Установка цен номенклатуры");
    • Фактические трудовые затраты - стоимость фактических затрат на трудовые ресурсы (поле "Стоимость работ" из документа "Акт о выполнении этапа работ");
    • Общие затраты - итоговая стоимость материальных и трудовых затрат, которая затрачена на устранение отказа (рассчитывается по формуле: Фактические материальные затраты + Фактические трудовые затраты);
    • Итоговая колонка - итоговые данные о количестве, длительности ремонта, фактических материальных, трудовых и общих затратах за весь указанный период отчета.
