Документ «Учет наработки оборудования»

Документ предназначен для регистрации данных о наработке объектов ремонта. Сведения о наработке оборудования главным образом используются при формировании графика ППР.

Чтобы документ стал доступен, необходимо включить функциональную опцию "Учет наработки оборудования".

В форме выбора показателя наработки выводятся следующие поля:

  • Показатель — наименование показателя наработки.
