Документ «Учет наработки оборудования»

Документ предназначен для регистрации данных о наработке объектов ремонта. Сведения о наработке оборудования главным образом используются при формировании графика ППР.

Чтобы документ стал доступен, необходимо включить функциональную опцию "Учет наработки оборудования".

В форме документа выводятся следующие поля:

  • Номер — номер документа. 
  • от — дата регистрации документа.
  • Организация — название организации, в которой производится регистрация данных о наработке оборудования.
  • Подразделение — наименование подразделения организации, в котором производится регистрация данных о наработке оборудования.
  • Комментарий — дополнительная текстовая информация о наработке оборудования.
  • Ответственный — ФИО ответственного за регистрацию данных о наработке.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Наработка оборудования»

  • N — номер записи о наработке объекта ремонта в списке регистрируемых значений. 
  • Положение — положение объекта ремонта в структуре иерархии.
  • Объект — наименование объекта ремонта, для которого регистрируется наработка. 
  • Показатель — обозначение показателя, в разрезе которого фиксируется наработка объекта ремонта. 
  • Ед. изм. — единица измерения, в которой учитывается наработка объекта ремонта.
  • Инв. № — инвентарный номер объекта ремонта.
  • Тех. № — технологический номер объекта ремонта.
  • Дата работы с — значение определяет начало периода, за который регистрируется наработка объекта ремонта.
  • Текущее значение — значение наработки на текущую дату (если ранее наработка не фиксировалась по данному показателю, то поле отображается незаполненным).
  • Наработка — значение параметра наработки, которое будет зарегистрировано для объекта ремонта.
  • Дата работы по – значение определяет окончание периода, за который регистрируется наработка объекта ремонта.
  • Новое значение — новое значение наработки объекта ремонта.
  • Распространять на подчиненные ОР — признак, определяющий возможность назначить подчинённому объекту ремонта ту же наработку, что и основному. При установленном признаке наработка, введённая объекту ремонта, автоматически присвоится и всем подчинённым объектам ремонта.
  • Иерархия — структура иерархии, в которой рассматривается данный объект ремонта.
  • Синхронизировать — если признак установлен, всем подчинённым объектам ремонта с помощью кнопки [Синхронизировать] можно автоматически присвоить значение наработки основного объекта ремонта. 

Кнопка [Подбор] предназначена для оперативного формирования списка объектов ремонта с помощью обработки "Подбор объектов ремонта".

С помощью кнопки [Заполнить по плановой наработке] осуществляется заполнение значений наработки выбранных объектов ремонта значениями плановой наработки.

С помощью кнопки [Заполнить по вышестоящему объекту ремонта] осуществляется заполнение значений наработки, соответствующих вышестоящему объекту ремонта.
