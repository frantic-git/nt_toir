Документ «Учет контролируемых показателей»

Документ предназначен для регистрации значений контролируемых показателей оборудования в системе. 

Документ можно ввести из рабочего места учета показателей эксплуатации, в журнале объекта ремонта на закладке "Показатели эксплуатации", при закрытии документов "Акт о выполнении регламентного мероприятия" и "Акт о выполнении этапа работ", а также ввести вручную непосредственно в списке.

Чтобы документ стал доступен, необходимо включить функциональную опцию "Учет контролируемых показателей".

В форме документа выводятся следующие поля:

  • Дата – дата документа в системе.
  • Номер – номер документа в системе.
  • Организация – название организации, в которой оформляется документ.
  • Подразделение – наименование подразделения организации, в котором оформляется документ.
  • Ответственный – ФИО ответственного за учет контролируемых показателей.
  • Комментарий – дополнительная текстовая информация о документе.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Оборудование»

  • Положение – положение объекта ремонта в структуре иерархии.
  • Оборудование – наименование объекта ремонта, для которого регистрируется значения контролируемых показателей.

Кнопка [Подбор] предназначена для оперативного формирования списка оборудования с помощью формы подбора.

Табличная часть «Контролируемые  показатели»

  • Дата контроля – дата учета контролируемого показателя.
  • Показатель – наименование контролируемого показателя.
  • Ед. изм. – единица измерения контролируемого показателя.
  • Значение – значение контролируемого показателя.
  • Исполнитель – ФИО сотрудника, зафиксировавшего значение контролируемого показателя.

Кнопка [Заполнить] предназначена для автоматического заполнения табличной части контролируемых показателей.
