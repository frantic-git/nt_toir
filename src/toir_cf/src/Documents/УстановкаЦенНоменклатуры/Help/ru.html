Документ «Установка цен номенклатуры»

Документ предназначен для фиксации цен на конкретные номенклатурные позиции. 

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • Дата – дата регистрации документа в системе.
  • Комментарий – дополнительная текстовая информация о документе.
  • Ответственный – ФИО пользователя, ответственного за документ.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Выбор видов цен»

Предназначена для выбора вида цены, в рамках которого будут устанавливаться цены номенклатуры. Видов цен можно выбрать несколько и для каждого установить свою цену номенклатуры. 

  • Вид цены – вид цены, в рамках которого устанавливается цена номенклатуры.
  • Цена – стоимость единицы номенклатуры в рамках вида цены.
По кнопке  [Перейти к установке цен] осуществляется переход к форме установки цен.

Табличная часть «Номенклатуры и цены»

Предназначена для установки цен номенклатуры.

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование номенклатуры.
  • Характеристика – характеристика номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры".
  • Вид цены – вид цены, в рамках которого устанавливается цена номенклатуры.
  • Цена – стоимость единицы номенклатуры в рамках вида цены.
  • Ед. изм. – единица измерения номенклатуры. 
Кнопка [Добавить] позволяет добавить номенклатуру в табличную часть.
Кнопка [Подбор] позволяет подобрать номенклатуру в табличную часть с помощью формы подбора.
Кнопка [Перейти к изменению состава видов цен] позволяет перейти к форме выбора вида цены, в рамках которого в документе будут устанавливаться цены номенклатуры.
