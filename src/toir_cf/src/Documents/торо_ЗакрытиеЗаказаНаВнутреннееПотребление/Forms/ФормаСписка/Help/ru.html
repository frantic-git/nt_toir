Документ «Закрытие заказа на внутреннее потребление» 

Документ предназначен для закрытия потребности по заказу на внутреннее потребление в случаях, когда для выполнения ремонта была использована не вся номенклатура, зарезервированная под заказ. Для создания документа рекомендуется использовать обработку "Рабочее место специалиста по обеспечению".

В форме списка выводятся следующие поля:

  • Дата – дата регистрации документа в системе.
  • Номер – номер документа в системе.
  • Заказ на внутреннее потребление – номер и дата заказа на внутреннее потребление, потребность по которому закрывается.
