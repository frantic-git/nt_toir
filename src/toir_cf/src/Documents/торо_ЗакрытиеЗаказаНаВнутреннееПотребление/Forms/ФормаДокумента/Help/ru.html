Документ «Закрытие заказа на внутреннее потребление» 

Документ предназначен для закрытия потребности по заказу на внутреннее потребление в случаях, когда для выполнения ремонта была использована не вся номенклатура, зарезервированная под заказ. Для создания документа рекомендуется использовать обработку "Рабочее место специалиста по обеспечению".

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Заказ на внутреннее потребление – номер и дата заказа на внутреннее потребление, потребность по которому закрывается. 
  • Организация – наименование организации, в которой производится резервирование.
  • Подразделение – наименование подразделения, в котором производится резервирование.
  • Комментарий – текстовый комментарий к документу.
  • Ответственный – ФИО пользователя, ответственного за документ.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Товары»

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование номенклатуры, по которой закрывается потребность.
  • Характеристика – характеристика номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры".
  • Серия – серия номенклатуры. Отображается при включенной функциональной опции "Использовать серии номенклатуры".
