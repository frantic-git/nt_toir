Документ «Установка минимального остатка номенклатуры»

Документ предназначен для установки значений минимального остатка номенклатуры, что позволяет вести контроль отрицательных остатков.

Чтобы вести контроль остатков номенклатуры при создании документов на внутреннее потребление и при перемещении товаров, необходимо включить функциональную опцию "Использовать контроль отрицательных остатков".

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Дата установки – дата установки минимального остатка.
  • Организация – наименование организации, в которой устанавливается минимальный остаток.
  • Склад – наименование склада, на котором устанавливается минимальный остаток.
  • Комментарий – дополнительная текстовая информация о документе.
  • Ответственный – ФИО пользователя, создавшего документ.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Список номенклатуры»

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование номенклатуры, для которой устанавливается минимальный остаток.
  • Ед. изм. – единицы измерения номенклатуры.
  • Мин. остаток – минимальное количество номенклатуры, которое допустимо на складе.
Кнопка [Добавить] позволяет добавить номенклатуру в табличную часть.
Кнопка [Подбор] позволяет подобрать номенклатуру в табличную часть с помощью формы подбора.
Кнопка [Рекомендуемый остаток по статистике потребления]* позволяет для указанной номенклатуры рассчитать значение минимального остатка. Доступны следующие варианты:
    • По максимуму за месяц - берется максимальное значение потребления за аналогичный предшествующий период
    • По минимуму за месяц - берется минимальное значение потребления за аналогичный предшествующий период;
    • По среднему за месяц - берется среднее значение потребления за аналогичные предшествующие периоды;
    • По прошлому году - берется общее потребление номенклатуры за предшествующий период;
    • По прошлому году с страховкой - берется общая статистика потребления номенклатуры за предшествующий период.
* Кнопка доступна при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов – МТО: Учитывать статистику потребления.
