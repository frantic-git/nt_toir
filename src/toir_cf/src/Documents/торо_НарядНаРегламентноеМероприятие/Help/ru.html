Документ «Наряд на регламентное мероприятие»

Документ предназначен для регистрации начала этапа работ по каждой операции, привязанной к графику регламентных мероприятий. 

Чтобы документ стал доступен, нужно включить функциональную опцию "Использовать наряды на регламентные мероприятия".

В форме документа выводятся следующие поля:

  • Номер – номер документа.
  • От – дата регистрации документа.
  • Организация – название организации предприятия, от имени которой оформляется наряд.
  • Подразделение – наименование подразделения, в котором оформлен наряд.
  • Комментарий – дополнительная текстовая информация о наряде.
  • Ответственный – ФИО пользователя, который создал наряд.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Закладка «Ремонтные работы»

Содержит информацию о списке объектов регламентных мероприятий и видах мероприятий в наряде.
  • N – порядковый номер записи в списке.
  • Список объектов – наименование списка объектов регламентных мероприятий, на выполнение мероприятий по которым оформлен наряд.
  • Вид мероприятия – вид регламентного мероприятия.
  • Дата начала – плановая дата и время начала ремонта.
  • Дата окончания – плановая дата и время окончания ремонта.

Кнопка [Подбор] предназначена для оперативного формирования списка мероприятий с помощью обработки "Подбор работ по ТОиР из документов-оснований". 

Подзакладка «Ремонтные работы»

Имеет иерархическую структуру и содержит перечень ремонтных работ и их технологических операций для списка объектов регламентных мероприятий.
  • Ремонтная работа – название работы, по которой выполняется наряд.
  • Количество –  количество ремонтов или технологических операций в наряде.
  • Ед. изм. – единица измерения операции.
Кнопка [Подбор] предназначена для оперативного формирования списка технологических операций.
Кнопка [Подбор по бригадам] позволяет подобрать исполнителей из сформированных бригад в соответствии с требуемыми квалификациями исполнителей.
Кнопка [Заполнить] позволяет заполнить исполнителей по нормам либо по основаниям.

Подзакладка «Список объектов»

Содержит список объектов регламентного мероприятия.
  • Объект ремонта – название объекта ремонта, входящего в список объектов регламентного мероприятия.

Подзакладка «Хозспособ»

Содержит список сотрудников-исполнителей работ. 

  • Сотрудник – ФИО сотрудника, который является исполнителем регламентного мероприятия по наряду. 
  • Квалификация – название квалификации сотрудника, являющегося исполнителем регламентного мероприятия.
  • Дата начала – плановая дата начала работы сотрудника по регламентному мероприятию.
  • Дата окончания – плановая дата окончания работы сотрудника по регламентному мероприятию.
  • Время работы – время работы сотрудника по регламентному мероприятию.

Кнопка [Подбор по бригадам] позволяет подобрать исполнителей из сформированных бригад в соответствии с требуемыми квалификациями исполнителей.

Кнопка [Заполнить исполнителей] предназначена для подбора исполнителей в документ.

Подзакладка «Подрядный»

Содержит список контрагентов-исполнителей работ. 

  • Контрагент – ФИО контрагента или наименование организации-контрагента, выполняющей регламентное мероприятие по наряду. 
  • Договор контрагента – договор, по которому контрагент будет выполнять мероприятие.
  • Дата начала – плановая дата начала работы контрагента по регламентному мероприятию.
  • Дата окончания – плановая дата окончания работы контрагента по регламентному мероприятию.

Закладка «Документы-основания»

Содержит список документов, на основании которых создан наряд.
  • N – порядковый номер записи в списке.
  • Документ-основание – наименование, номер и дата документа-основания.

Закладка «Дополнительно»

  • Комиссия – поле выбора комиссии, члены которой будут вписаны в печатную форму наряда. Становится доступна, если включена функциональная опция "Использовать комиссии".
