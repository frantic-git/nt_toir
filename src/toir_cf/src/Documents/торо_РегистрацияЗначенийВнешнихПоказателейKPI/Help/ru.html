Документ «Регистрация значений внешних показателей»

Документ предназначен для регистрации в системе значений показателей процессов (KPI), которые нельзя получить из данных, имеющихся в ТОиР.

Документ регистрации значений внешних показателей становится доступен при включении функциональной опции "Показывать подсистему "Мониторинг показателей процессов". 

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Организация – наименование организации предприятия, в которой оформляется документ.
  • Подразделение  – наименование подразделения организации, в котором оформляется документ.
  • Комментарий – дополнительная текстовая информация о документе.
  • Ответственный – ФИО пользователя, ответственного за документ.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Значения показателей»

  • N – порядковый номер записи в списке.
  • Показатель – наименование показателя процессов (KPI), значение которого регистрируется.
  • Период – период, на который получено значение.
  • Значение – значение показателя.
  • Подразделение – подразделение, для которого действует значение.
  • Объект ремонта – объект ремонта, для которого действует значение.
