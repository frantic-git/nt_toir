Документ «Установка аналогов номенклатуры для ремонтов»

Документ предназначен для установки возможности замены номенклатуры на аналогичную при ремонте.

Использование аналогов номенклатуры становится доступным при включении функциональной опции "Использовать аналоги номенклатуры для ремонтов". 

В форме списка выводятся следующие поля:

  • Дата – дата регистрации документа в системе.
  • Номер – номер документа в системе.
  • Организация – наименование организации, для которой в документе установлено использование аналогов.
  • Подразделение – наименование подразделения, для которого в документе установлено использование аналогов.
  • Объект ремонта – наименование объекта ремонта, для которого в документе установлено использование аналогов.
  • Вид ремонта – вид ремонта, для которого в документе установлено использование аналогов.
  • Указание по применению – текстовые указания по использованию аналогов.
  • Действует с – дата, начиная с которой действует использование аналогов из документа.
  • Действует по – последний день действия аналогов из документа.
  • Ответственный – ФИО пользователя, ответственного за установку аналогов.
