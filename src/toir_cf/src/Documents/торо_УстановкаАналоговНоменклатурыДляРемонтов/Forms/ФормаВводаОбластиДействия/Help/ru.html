Документ «Установка аналогов номенклатуры для ремонтов»

Документ предназначен для установки возможности замены номенклатуры на аналогичную при ремонте.

Использование аналогов номенклатуры становится доступным при включении функциональной опции "Использовать аналоги номенклатуры для ремонтов". 

В форме ввода области действия выводятся следующие отборы:
  • Для всех объектов ремонта – установленные аналоги будут действовать для всех объектов ремонта.
  • Группа объектов ремонта –  установленные аналоги будут действовать для группы объектов ремонта.
  • Организация –  установленные аналоги будут действовать для объектов ремонта из указанной организации.
  • Направление – установленные аналоги будут действовать для объектов ремонта из указанного направления объектов ремонта.
  • Объект ремонта – установленные аналоги будут действовать для указанного объекта ремонта.
  • Для всех ремонтов – установленные аналоги будут действовать для всех ремонтов.
  • Подразделение-исполнитель –  установленные аналоги будут действовать для всех ремонтов указанного подразделения-исполнителя.
  • Вид ремонта – установленные аналоги будут действовать для указанного вида ремонта.
  • Технологическая карта – установленные аналоги будут действовать для указанной технологической карты.
