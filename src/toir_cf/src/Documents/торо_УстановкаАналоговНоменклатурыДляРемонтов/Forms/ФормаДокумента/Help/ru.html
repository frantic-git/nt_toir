Документ «Установка аналогов номенклатуры для ремонтов»

Документ предназначен для установки возможности замены номенклатуры на аналогичную при ремонте.

Использование аналогов номенклатуры становится доступным при включении функциональной опции "Использовать аналоги номенклатуры для ремонтов". 

В форме документа выводятся следующие поля:
  • Номер – номер документа в системе.
  • Дата – дата регистрации документа в системе
  • Период действия с – дата, начиная с которой действует использование аналогов из документа.
  • По – последний день действия аналогов из документа.
  • Действие – гиперссылка на таблицу, в которой можно сузить использование аналогов до организации, направления, объекта или группы объектов ремонта, а также подразделения-исполнителя, вида ремонта и технологической карты.
  • Ответственный – ФИО пользователя, ответственного за установку аналогов.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.
  • Указание по применению – текстовые указания по использованию аналогов.
Кнопка [Установить интервал] позволяет выбрать период действия аналогов.

Табличная часть «Набор заменяемых материалов»

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование заменяемой номенклатуры.
  • Характеристика – характеристика заменяемой номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры".
  • Ед. изм. – единицы измерения заменяемой номенклатуры.
  • Количество– количество единиц заменяемой номенклатуры.

Табличная часть «Набор, аналогичный набору заменяемых материалов»

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование номенклатуры-аналога.
  • Характеристика – характеристика номенклатуры-аналога. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры".
  • Ед. изм. – единицы измерения номенклатуры-аналога.
  • Количество– количество единиц номенклатуры-аналога.
Кнопка [Подбор] позволяет подобрать номенклатуры в табличные части с помощью формы подбора.
Кнопка [Добавить] позволяет добавить номенклатуры в табличные части.
