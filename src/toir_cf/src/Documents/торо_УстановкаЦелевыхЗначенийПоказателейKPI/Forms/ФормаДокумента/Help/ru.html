Документ «Установка целевых значений показателей (KPI)»

Документ предназначен для ввода целевых значений показателей, которые в дальнейшем будут использоваться при нормировании.

Документ установки целевых значений показателей (KPI) становится доступен при включении функциональный опции "Показывать подсистему "Мониторинг показателей процессов". 

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Организация – наименование организации, от имени которой оформляется документ.
  • Комментарий – дополнительный текстовый комментарий к документу.
  • Ответственный – ФИО пользователя, ответственного за установку аналогов.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Ключевые показатели»

  • N – порядковый номер записи в списке.
  • Показатель – наименование показателя, значение которого устанавливается.
  • Организация – организация, для которой действует значение.
  • Подразделение – подразделение, для которого действует значение.
  • Объект ремонта – объект ремонта, для которого действует значение.
  • Значение показателя – целевое значение, которого необходимо достигнуть или превысить.
  • Критическое значение показателя –  значение, ниже которого состояние показателя считается критическим. Можно задать критическое значение показателя больше значения показателя, тогда смысл изменится соответственно: критическими будут считаться значения выше критического значения.
  • Значение показателя ограничено по времени – отметка, установка которой включает ограничение периода действия значения.
  • Дата по – дата, по которую действует значение показателя.
