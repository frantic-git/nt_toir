Документ «Выбытие объекта ремонта»

Документ предназначен для документального оформления списания объектов ремонта.

Для использования документа принятия объекта ремонта к учету необходимо включить функциональную опцию "Использовать документы принятия и списания оборудования".

В форме документа выводятся следующие поля:

  • Номер – номер документа выбытия.
  • От – дата регистрации документа выбытия.
  • Организация – наименование организации предприятия, в которой фиксируется выбытие объекта ремонта.
  • Подразделение  – наименование подразделения организации, в котором фиксируется выбытие объекта ремонта.
  • Объект ремонта – наименование выбывающего объекта ремонта.
  • Дата списания – дата, начиная с которой объект ремонта будет считаться списанным. 
  • Склад –  склад для списываемого объекта ремонта.
  • Комментарий – дополнительная текстовая информация о документе выбытия.
  • Ответственный – ФИО пользователя, который является ответственным за выбытие объекта ремонта.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Закладка «Общие сведения»

Содержит информацию об инвентарном номере объекта ремонта, дате его ввода в эксплуатацию и его положении в структуре иерархии объектов ремонта. Структуру иерархии можно менять, нажав на гиперссылку с названием иерархии.

  • Инвентарный № – инвентарный номер объекта ремонта.
  • Дата ввода в эксплуатацию – дата ввода из документа "Принятие объекта ремонта к учету".
  • Входил в состав – вышестоящий объект ремонта. 

Закладка «Подчиненные объекты ремонта»

Если у объекта ремонта есть подчиненный объект, то на этой закладке будет информация о нем. Чтобы увидеть подчиненные объекты ремонта, нужно нажать на кнопку [Обновить подчиненные].

  • N – номер элемента списка.
  • Снять с учета – отметка, позволяющая снять с учета в одном документе с основным и подчиненные ему объекты ремонта.
  • Объект ремонта – наименование подчиненного объекта ремонта.

Закладка «Комиссия»

Данная закладка содержит в себе список сотрудников, которые войдут в комиссию по фиксированию выбытия объекта ремонта и будут указаны на печатной форме документа выбытия.

  • N – номер элемента списка.
  • Сотрудник – ФИО ответственного сотрудника.

Закладка «Номенклатура»

Данная закладка содержит информацию о запчастях списываемого объекта ремонта.

  • N – номер элемента списка.
  • Номенклатура – наименование номенклатуры.
  • Количество – количество номенклатуры.
  • Склад – наименование склада номенклатуры.
