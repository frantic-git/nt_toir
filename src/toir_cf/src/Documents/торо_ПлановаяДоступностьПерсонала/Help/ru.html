Документ «Плановая доступность персонала»

Документ предназначен для регистрации плановой доступности персонала. Предполагается не ручной пользовательский ввод, а использование рабочего места "Управление плановой доступностью персонала", которое автоматически создаёт данные документы.

В форме списка выводятся следующие поля:

  • Дата – дата регистрации документа в системе.
  • Номер – номер документа в системе.
  • Организация – наименование организации, для сотрудников которой указывается доступность персонала.
  • Подразделение – наименование подразделения организации, для сотрудников которого указывается доступность персонала.
  • Начало планирования – дата начала планирования доступности.
  • Окончание планирования – дата окончания планирования доступности.
  • Комментарий – дополнительная текстовая информация о документе.
  • Ответственный – ФИО пользователя, ответственного за документ плановой доступности.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.
