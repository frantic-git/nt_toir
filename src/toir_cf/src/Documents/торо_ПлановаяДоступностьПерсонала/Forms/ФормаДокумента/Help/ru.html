Документ «Плановая доступность персонала»

Документ предназначен для регистрации плановой доступности персонала. Предполагается не ручной пользовательский ввод, а использование рабочего места "Управление плановой доступностью персонала", которое автоматически создаёт данные документы.

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Период планирования – период планирования доступности.
  • Организация – наименование организации, для сотрудников которой указывается доступность персонала.
  • Подразделение – наименование подразделения, для сотрудников которого указывается доступность персонала.
  • Ответственный – ФИО пользователя, ответственного за документ плановой доступности.
  • Комментарий – дополнительная текстовая информация о документе.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Раздел «Рабочее время сотрудников»

  • N — порядковый номер записи о сотруднике в списке.
  • Сотрудник — ФИО сотрудника организации, для которого указывается доступность.
  • Должность — должность сотрудника, для которого указывается доступность.
  • Дата графика — дата, на которую указывается доступность сотрудника.
  • Состояние сотрудника — значение состояния доступности сотрудника.
  • Время начала — время начала действия состояния сотрудника.
  • Время окончания — время окончания действия состояния сотрудника.
