Документ «Заказ поставщику»

Документ предназначен для заказа недостающих активов у поставщика, для поддержания складского запаса, или при наличии прочей необходимости. 

Документ может быть создан вручную из формы списка документов, а также в рабочем месте специалиста по обеспечению. На основании заказа поставщику создается документ "Поступление товаров и услуг", в котором фиксируется факт поступления заказа.

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе.
  • От – дата регистрации документа в системе.
  • Дата поставки – наименьшая дата потребности, по которой создается заказ.
  • Организация – наименование организации, в которой производится заказ поставщику.
  • Подразделение – наименование подразделения, в котором производится заказ поставщику.
  • Контрагент – поставщик, у которого в документе осуществляется заказ.
  • Договор – наименование договора с поставщиком.
  • Склад – предпочтительный склад поступления заказа.
  • Валюта – валюта, в которой вычисляется сумма заказа.
  • Цена включает НДС – отметка, при выборе которой в цене номенклатуры изначально учитывается НДС, а в табличной части "Товары" будет отсутствовать реквизит  "Сумма с НДС".
  • Комментарий – дополнительная текстовая информация о заказе поставщику.
  • Ответственный – ФИО пользователя, ответственного за регистрацию заказа поставщику.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Товары»

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование номенклатуры.
  • Характеристика – характеристика номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры".
  • Ед. изм. – вид единицы измерения номенклатуры.
  • Количество – количество единиц номенклатуры.
  • Цена – стоимость единицы номенклатуры.
  • Сумма – общая стоимость всего количества единиц номенклатуры.
  • Ставка НДС – ставка НДС в процентах.
  • НДС – сумма НДС по номенклатуре в валюте документа.
  • Сумма с НДС – общая стоимость всего количества единиц номенклатуры с НДС; отображается при отключенной настройке "Цены включает НДС".
  • Склад – склад поступления.
Кнопка [Добавить] позволяет добавить номенклатуру в табличную часть.
Кнопка [Подбор] позволяет подобрать номенклатуру в табличную часть с помощью формы подбора.
Кнопка [Рассчитать количество с учетом статистики потребления]* позволяет для указанной номенклатуры рассчитать количество для заказа поставщику. Доступны следующие варианты:
    • По максимуму за месяц - берется максимальное значение потребления за аналогичный предшествующий период
    • По минимуму за месяц - берется минимальное значение потребления за аналогичный предшествующий период;
    • По среднему за месяц - берется среднее значение потребления за аналогичные предшествующие периоды;
    • По прошлому году - берется общее потребление номенклатуры за предшествующий период;
    • По прошлому году с страховкой - берется общая статистика потребления номенклатуры за предшествующий период.
* Кнопка доступна при включенном флаге в разделе Настройка параметров системы - Интерфейс и ввод документов – МТО: Учитывать статистику потребления.
