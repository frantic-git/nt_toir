Документ «Ввод начальных данных»

Документ предназначен для ввода начальных данных для объектов ремонта. По каждому объекту ремонта, включенному в документ, можно задать значения измеряемых показателей, информацию о последних ремонтных работах, а также сведения о значениях показателей наработки. Первоначальные данные вводятся в систему в процессе ее внедрения на этапе ввода НСИ.

При создании по объекту ремонта в форме документа выводятся следующие поля:
  • Номер – номер документа.
  • От – дата регистрации документа.
  • Организация – наименование организации, регистрирующей документ.
  • Подразделение – наименование подразделения организации, регистрирующей документ.
  • Ответственный – ФИО пользователя, создавшего документ. 
  • Комментарий – текстовый комментарий к документу. 
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Объекты ремонта»

  • N – порядковый номер записи в списке. 
  • Положение – текущее положение вводимого объекта ремонта.
  • Объект ремонта – оборудование, для которого производится ввод начальных данных.
  • Инвентарный номер – инвентарный номер оборудования.
  • Распространять на подчиненные ОР – при установленном значении позволяет распространить введенные данные на все объекты ремонта, находящиеся в иерархии к текущему.
  • Структура иерархии – структура иерархии, которую необходимо использовать при распространении данных на подчиненные объекты ремонта.
  • Начальное состояние – вид эксплуатации объекта ремонта.

Закладка «Выполненные работы»

  • Вид ремонтных работ – ссылка на последний ремонт, который проводился для оборудования. 
  • Дата окончания ремонта – дата и время окончания последнего ремонта, который проводился для оборудования.

Закладка «Показатели»

  • Дата контроля — дата и время проведения контроля измеряемых показателей оборудования. 
  • Показатель — название измеряемого показателя. 
  • Значение — регистрируемое значение измеряемого показателя.

Закладка «Наработка»

  • Показатель — название показателя наработки. 
  • Дата работы по — дата регистрации значения показателя наработки оборудования.
  • Значение — регистрируемое значение показателя наработки оборудования.
  • Итог на дату — итоговое значение наработки на дату регистрации значения.

Закладка «Запчасти»

  • Номенклатура – наименование номенклатуры.
  • Серийный номер – серийный номер запчасти.
  • Статус движения запчасти – вид замены запчасти. Значение в поле выбирается из встроенного списка и носит справочное значение.
  • Кол-во – количество указанных запчастей в оборудовании.
При создании по списку объектов ремонта в форме документа выводятся поля:
  • Номер – номер документа.
  • От – дата регистрации документа.
  • Организация – наименование организации, регистрирующей документ.
  • Подразделение – наименование подразделения организации, регистрирующей документ.
  • Комментарий – текстовый комментарий к документу. 
  • Ответственный – ФИО пользователя, создавшего документ. 

Табличная часть «Список объектов ремонта»

  • N – порядковый номер записи в списке. 
  • Список объектов – наименование списка объектов регламентных мероприятий.

Табличная часть «Мероприятия»

  • Вид мероприятия – вид регламентного мероприятия. 
  • Дата последнего мероприятия – дата последнего регламентного мероприятия.
