Документ «График регламентных мероприятий»

Документ предназначен для формирования графика выполнения регламентных мероприятий для списков объектов регламентных мероприятий (маршрутов), зарегистрированных в системе, на указанный период.

В форме документа выводятся следующие поля:

  • Номер – номер документа.
  • От – дата регистрации документа.
  • Дата планирования – дата начала планирования.
  • Продолжительность – количество периодов, на которое сформирован график, и их вид.
  • Организация – название организации предприятия, от имени которой формируется/корректируется график.
  • Подразделение – наименование подразделения, в котором формируется/корректируется график.
  • Уточняемый график – отображается только для корректировки графика, содержит ссылку на корректируемый график регламентных мероприятий.
  • Комментарий – дополнительная текстовая информация о графике/корректировке.
  • Ответственный – ФИО пользователя, создавшего/скорректировавшего график.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Закладка «План-график регламентных мероприятий»

Содержит информацию о списках объектов регламентных мероприятий, видах мероприятий, которые запланировано провести, их графике, исполнителях, обеспечении и стоимости.

Табличная часть «Объекты регламентных мероприятий»

  • Список объектов / Вид мероприятия / Объект ремонта – наименование списка объектов регламентных мероприятий, для которого формируется/корректируется график, вида регламентного мероприятия и объектов ремонта в списке.
  • Инвентарный номер – инвентарный номер объекта ремонта.

Кнопка [Подбор] предназначена для оперативного формирования списка объектов регламентного мероприятия.

Кнопка [Заполнить] предназначена для расчета графика регламентных мероприятий для всех объектов или выборочно. Рассчитать график можно только после записи документа.

  • Список объектов ремонта – название списка объектов регламентного мероприятия, для которого запланировано мероприятия.
  • Вид мероприятия – вид регламентного мероприятия.
  • Период – даты начала и окончания мероприятия.
  • Новый период – отображается только для корректировки графика, новые даты начала и окончания мероприятия при его переносе.
  • Срок по нормативу – точная дата начала мероприятия, рассчитанная по нормативам без учета графика выполнения ремонтных работ.
  • Способ выполнения – способ выполнения регламентного мероприятия.
  • Исполнитель – исполнитель регламентного мероприятия.
  • Отменен – отображается только для корректировки графика, признак отмененного мероприятия.
  • Склад – склад обеспечения мероприятия.
  • Стоимость мероприятия – плановая стоимость мероприятия.
Кнопка  позволяет отобразить замещенные мероприятия.
Кнопка [Заполнить] предназначена для заполнения исполнителей и стоимостей ремонта.

Табличная часть «Заказы на внутреннее потребление»

  • Заказ на внутреннее потребление – номер и дата заказа на внутреннее потребление, созданного для обеспечения мероприятия.
  • Дата отгрузки – плановая дата отгрузки заказа.
  • Склад – предпочтительный склад отгрузки заказа.

Закладка «Дополнительно»

Содержит информацию о версии ППР, на которой рассчитывается график.

Закладка «Диаграмма ремонтов»

Представляет собой визуальное отображение графика регламентных мероприятий. Перемещение "ползунка" в горизонтальном направление позволит увеличить/уменьшить масштаб отображения диаграммы для наиболее оптимального варианта визуального представления графика.

С помощью кнопки [Построить/Обновить] можно вывести графическое представление графика регламентных мероприятий.
