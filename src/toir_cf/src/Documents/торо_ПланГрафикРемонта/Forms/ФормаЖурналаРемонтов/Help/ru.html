Журнал запланированных ремонтов

Журнал вызывается из формы списка документов "План-график ППР" или из формы документа "План-график ППР" нажатием кнопки [Печать - Журнал запланированных ремонтов].

Журнал предназначен для отображения информации по запланированным ремонтам. 

Для всех указанных нормативных ремонтов указываются следующие данные: Объект ремонта/Вид ремонта/Технологические операции ремонта; Дата ремонта/Описание; Время выполнения; Материальные затраты (количество); Исполнитель; Подпись; Отметка о выполнении.

В форме выводятся следующие поля:

  • Период – период, на который будет выведена информация по запланированным ремонтам.
  • Объект ремонта – объект ремонта, по которому в журнал будет выведена информация по запланированным ремонтам.
  • Документ – дата и номер план-графика ППР, для которого формируется журнал.
Кнопка [Сформировать] предназначена для формирования журнала запланированных ремонтов.
