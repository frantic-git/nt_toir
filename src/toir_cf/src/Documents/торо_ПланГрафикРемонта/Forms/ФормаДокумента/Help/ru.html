Документ «План-график ППР»

Документ предназначен для формирования графика ППР (планово-предупредительного ремонта) для объектов ремонта, зарегистрированных в системе, на указанный период.

Чтобы документ стал доступен, нужно включить функциональную опцию "Использовать ППР".

В форме документа выводятся следующие поля:

  • Номер – номер документа.
  • От – дата регистрации документа.
  • Дата планирования – дата начала планирования.
  • Продолжительность – количество периодов, на которое сформирован график, и их вид.
  • Организация – название организации предприятия, от имени которой формируется/корректируется график.
  • Подразделение – наименование подразделения, в котором формируется/корректируется график.
  • Организация-владелец – объекты ремонта, принадлежащие указанной в этом поле организации, могут быть добавлены в документ.
  • Подразделение-владелец – объекты ремонта, принадлежащие указанному в этом поле подразделению, могут быть добавлены в документ.
  • Уточняемый график – отображается только для корректировки графика, содержит ссылку на корректируемый график.
  • Комментарий – дополнительная текстовая информация о графике/корректировке.
  • Ответственный – ФИО пользователя, создавшего/скорректировавшего график.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Закладка «План-график ППР»

Содержит информацию об объектах ремонта, видах ремонтов, которые запланировано провести, их графике, исполнителях, обеспечении и стоимости.

Табличная часть «Объекты ремонта»

  • Положение/Объект ремонта/Вид ремонта – положение объекта ремонта в структуре иерархии, наименование объекта ремонта и вид нормативного ремонта, для которого формируется/корректируется график.
  • Инвентарный номер – инвентарный номер объекта ремонта.
  • № п.п. – порядковый номер. 

Кнопка [Подбор] предназначена для оперативного формирования списка объектов ремонта.

Кнопка [Корректировать ППР по лимитам] открывает обработку "Корректировка ППР по РМ". 

Кнопка [Заполнить] предназначена для расчета план-графика ППР для всех объектов или выборочно. Рассчитать график можно только после записи документа.

Табличная часть «План ремонтов»

  • Вид  ремонтных работ – вид нормативного ремонта.
  • Период – даты начала и окончания ремонта.
  • Новый период – отображается только для корректировки графика, новые даты начала и окончания ремонта при его переносе.
  • Срок по нормативу – точная дата начала ремонта, рассчитанная по нормативам без учета графика выполнения ремонтных работ.
  • Способ выполнения – способ выполнения ремонта.
  • Исполнитель – исполнитель ремонта: подразделение или контрагент.
  • Отменен – отображается только для корректировки графика, признак отмененного ремонта.
  • Склад – склад обеспечения ремонта. По умолчанию заполняется значением из поля "Склад обеспечения МТР", если заполнено подразделение-исполнитель ремонта – значением поля "Склад" из карточки подразделения.
  • Замещен – признак замещенного ремонта.
  • Сумма ремонта – плановая стоимость ремонта.
Кнопка  позволяет отобразить замещенные ремонты.
Кнопка [Заполнить] предназначена для заполнения исполнителей и стоимостей ремонта.
Кнопка [Создать] позволяет создать заказы на внутреннее потребление на выбранный период.

Табличная часть «Заказы на внутреннее потребление»

  • Заказ на внутреннее потребление – номер и дата заказа на внутреннее потребление, созданного для обеспечения ремонта.
  • Дата отгрузки – плановая дата отгрузки заказа.
  • Склад – предпочтительный склад отгрузки заказа.

Закладка «Дополнительно»

Содержит информацию о версии ППР, на которой рассчитывается график.

Закладка «Диаграмма ремонтов»

Представляет собой визуальное отображение план-графика ППР. Перемещение "ползунка" в горизонтальном направление позволит увеличить/уменьшить масштаб отображения диаграммы для наиболее оптимального варианта визуального представления графика.

С помощью кнопки [Построить/Обновить] можно вывести графическое представление план-графика ППР
