Документ «Установка коэффициентов перевода стоимостей ремонтов»

Документ предназначен для ведения списка документов, устанавливающих коэффициенты перевода стоимостей ремонтов по подрядчикам для объектов ремонта.

В форме списка выводятся следующие поля:

  • Дата – дата регистрации документа в системе.
  • Номер – номер документа в системе.
  • Организация – наименование организации, в которой сформирован документ.
  • Вид операции – вид установки: по объекту или по списку объектов ремонта.
  • Ответственный – ФИО пользователя, ответственного за документ.
Кнопка [Создать / Документ по объекту ремонта] позволяет установить коэффициент перевода стоимостей ремонта для объектов ремонта.
Кнопка [Создать / Документ по списку объектов ремонта] позволяет установить коэффициент перевода стоимостей для списка объектов регламентных мероприятий.
