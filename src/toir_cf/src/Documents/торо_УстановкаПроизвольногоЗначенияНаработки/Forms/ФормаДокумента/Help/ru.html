Документ «Установка произвольного значения наработки»

Документ предназначен для установки произвольного значения наработки при обнулении счетчика, когда значение наработки достигает максимума, заданного нами при настройке параметров наработки для объекта ремонта, а также для установки произвольного значения наработки при распространении наработки на подчиненные объекты ремонта и синхронизации периодов наработки родителя и подчиненных. 

В первом случае документ создается автоматически, во втором – после установки отметки "Распространять на подчиненные ОР" в табличной части документа "Учет наработки оборудования" и нажатия кнопки "Синхронизировать".

Также документ используется, если нужно установить значение наработки, равное нулю

В форме документа выводятся следующие поля:

  • Номер – номер документа в системе. 
  • От – дата регистрации документа в системе. 
  • Номер – номер документа в системе. 
  • Организация – организация, в которой производится установка произвольного значения наработки.
  • Подразделение – наименование подразделения организации, в котором производится установка произвольного значения наработки. 
  • Комментарий – текстовый комментарий к документу.
  • Ответственный – ФИО сотрудника, ответственного за документ.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Табличная часть «Направления объектов ремонта»

  • N – порядковый номер записи в списке.
  • Положение – положение объекта ремонта в структуре иерархии.
  • Объект ремонта – наименование объекта ремонта, для которого регистрируется наработка.
  • Показатель – обозначение показателя, в разрезе которого фиксируется наработка объекта ремонта. 
  • Единица измерения – единица измерения, в которой учитывается наработка объекта ремонта.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • Технологический № – технологический номер объекта ремонта.
  • Дата последней наработки – начало периода, за который регистрируется наработка объекта ремонта.
  • Зафиксированная наработка – значение наработки на дату выполнения операции установки произвольного значения (если ранее наработка не фиксировалась по данному показателю, то поле отображается незаполненным).
  • Наработка – значение параметра наработки с даты последней наработки до даты операции установки произвольного значения, которое будет зарегистрировано для объекта ремонта.
  • Дата и время операции – значение определяет окончание периода, за который регистрируется наработка объекта ремонта (значение и начала, и окончания периода при сбросе показаний счетчика).
  • Показания счетчика на момент сброса – значение параметра наработки до выполнения операции установки произвольного значения  (сброса счетчика).
  • Абсолютная наработка – значение параметра наработки объекта ремонта за весь период его использования.
  • Начальное показание счетчика – значение параметра наработки после выполнения операции установки произвольного значения.
  • Причина изменения наработки – причина, по которой необходимо установить произвольное значение счетчика (выполнить сброс счетчика).
  • Распространять на подчиненные ОР – при установленном значении позволяет распространить введенные данные на все объекты ремонта, находящиеся в иерархии к текущему.
  • Структура иерархии – структура иерархии, которую необходимо использовать при распространении данных на подчиненные объекты ремонта.
  • Синхронизировать – отметка для синхронизации данных с подчиненными объектами ремонта, устанавливается после установки отметки "Распространять на подчиненные ОР".
Кнопка [Синхронизировать] предназначена для выполнения синхронизации данных с подчиненными объектами ремонта после установки отметки "Распространять на подчиненные ОР".
