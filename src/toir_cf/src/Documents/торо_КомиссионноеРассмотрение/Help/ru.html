Документ «Закупки (Лоты)»

Документ предназначен для ведения списка лотов, в которых выполняется выбор оптимального по совокупности критериев подрядчика.

Возможность использовать лоты включается с помощью функциональной опции "Использовать лоты", после чего в документах "Наряд на выполнение ремонтных работ" и "Акт о выполнении этапа работ" на закладке "Исполнители" становится активной кнопка "Заполнить из лотов" для подбора подрядчика, выполняющего ремонт.

В форме документа выводятся следующие поля:

  • Номер — номер документа.
  • От — дата и время регистрации документа в системе.
  • Дата проведения — дата проведения лота.
  • Организация — наименование организации, в которой сформирован документ.
  • Подразделение — наименование подразделения, в котором сформирован документ.
  • Наименования лота — название проводимого лота. 
  • Плановая сумма — плановая сумма лота.
  • План-график ППР — ссылка на документ "План-график ППР", для подрядных работ которого необходимо выбрать подрядчика. При заполнении в табличную часть "Ремонты" добавляются записи о ремонтах, сумма работ по которым не превышает значение, заданное в поле "Плановая сумма".
  • Сумма ремонтов — фактическая сумма лота.
  • Подрядчик — выбранный подрядчик.
  • Договор — наименование договора с подрядчиком.
  • Комментарий — дополнительная текстовая информация о документе.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
  • Автор – ФИО пользователя, создавшего документ. Недоступен для редактирования.

Закладка «Ремонты»

  • N – порядковый номер записи в списке.
  • Объект ремонта – название объекта ремонта. 
  • Вид ремонта — наименование вида обслуживания оборудования, по которому будут выполнены ремонтные работы.
  • Дата начала – дата, на которую было запланировано начало выполнения ремонтных работ.
  • Дата окончания – дата, на которую было запланировано окончание ремонтных работ.
  • Сумма ремонта – плановая сумма выполнения ремонта.
По кнопке [Заполнить] можно подобрать ремонтные работы из план-графика или заполнить их по нему. 

Закладка «Критерии выбора подрядчиков»

  • N – порядковый номер записи в списке.
  • Критерий – критерий, по которому выполняется выбор для выполнения ремонта.
  • Единица измерения – единица, в которой измеряется показатель.
  • Показатель – наименование показателя.

Закладка «Данные по кандидатам»

  • Кандидат – название кандидата.
  • Критерий – критерий, по которому выполняется выбор для выполнения ремонта.
  • Единица измерения – единица, в которой измеряется показатель.
  • Показатель – наименование показателя.

Закладка «Диаграмма выбора»

  • N – порядковый номер записи в списке.
  • Критерий – критерий, по которому выполняется выбор для выполнения ремонта.
  • Показатель – наименование показателя.
  • Данные по кандидатам - графическая диаграмма.
