Документ «Закупки (Лоты)»

Документ предназначен для ведения списка лотов, в которых выполняется выбор оптимального по совокупности критериев подрядчика.

Возможность использовать лоты включается с помощью функциональной опции "Использовать лоты", после чего в документах "Наряд на выполнение ремонтных работ" и "Акт о выполнении этапа работ" на закладке "Исполнители" становится активной кнопка "Заполнить из лотов" для подбора подрядчика, выполняющего ремонт.

В форме списка выводятся следующие поля:

  • Дата — дата и время регистрации документа в системе.
  • Номер — номер документа.
  • Организация — наименование организации, в которой сформирован документ.
  • Подразделение — наименование подразделения, в котором сформирован документ.
  • Подрядчик — выбранный подрядчик.
  • Договор — наименование договора с подрядчиком.
  • План-график ППР — ссылка на документ "План-график ППР", для подрядных работ которого необходимо выбрать подрядчика.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
  • Комментарий — дополнительная текстовая информация о документе "Закупки (Лоты)".
