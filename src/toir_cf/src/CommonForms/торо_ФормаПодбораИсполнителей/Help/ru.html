Форма подбора исполнителей

Форма подбора исполнителей по бригадам позволяет быстро заполнить поля сотрудников по квалификациям. Для работы формы необходимо, чтобы в документе уже были добавлены строки с указанием необходимых квалификаций.
Порядок работы с формой:
1. Выбрать бригаду.
2. В правой части окна со списком сотрудников в бригаде дважды кликнуть по сотруднику. Сотрудник подставится в строку с соответствующей ему квалификацией.
3. Нажать кнопку "Передать в документ" для переноса результата подбора в документ.
