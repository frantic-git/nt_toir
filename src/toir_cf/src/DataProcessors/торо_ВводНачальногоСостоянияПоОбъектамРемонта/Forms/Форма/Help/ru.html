Обработка «Ввод начального состояния по объектам ремонта»

Обработка предназначена для ввода документов "Состояния объектов ремонта" по тем объектам ремонта, которые были введены в эксплуатацию, но которым не установлено никакое состояние. Это необходимо для корректного расчета планов-графиков ППР.


В форме выводятся следующие поля для отбора:

Группа «Параметры поиска»

  • Организация – отбор по организации-владельцу объекта ремонта.
  • Подразделение – отбор по подразделению-владельцу объекта ремонта.
  • Только с настроенными нормативами – отбор по объектам ремонта у которых настроены нормативные ремонты для расчета планов-графиков ППР. Рекомендуется устанавливать этот флаг, если цель использования обработки только подготовка объектов ремонта к расчету ППР.

Группа «Параметры заполнения документов»

  • Организация – организация, от имени которой оформляется документ.
  • Подразделение – подразделение, в котором оформляется документ.
  • Вид эксплуатации – вид эксплуатации, который будет указан в документе "Состояние объектов ремонт" .

Кнопка [Найти объекты ремонта] позволяет заполнить ТЧ "Объекты ремонта".

Кнопка [Создать документы учеты состояний] позволяет ввести документы "Состояния объектов ремонта".

Табличная часть «Без состояний»

Предназначена для вывода данных об объектах ремонта без состояний.
  • № – порядковый номер строки.
  • Объект ремонта – наименование объекта ремонта.
  • Инвентарный номер – инвентарный номер объекта ремонт.
  • Заводской номер – номер изделия–объекта ремонта, установленный на заводе по номенклатуре объектов.
  • Типовой ОР – наименование типового объекта ремонта (ремонтной группы), в которую входит объект ремонта (отображается, если включена настройка "Использовать типовые объекты ремонта" в разделе Настройка параметров системы - Интерфейс и ввод документов - Оборудование и нормативы).
  • Положение – положение объекта ремонта в иерархии.
  • Есть нормативы – отметка о наличии у объекта ремонта нормативов (вкладка "Нормативы планирования - Нормативные ремонты" справочника "Объекты ремонта").
  • Статус в учете - статус объекта ремонта в учете (отображается, если включена настройка "Использовать документы принятия и списания оборудования" в разделе Настройка параметров системы - Интерфейс и ввод документов - Оборудование и нормативы).
  • Дата ввода в эксплуатацию – дата ввода в эксплуатацию объекта ремонта.
  • Ввести состояние на – дата ввода состояния объекта ремонта. По умолчанию используется дата ввода объекта ремонта в эксплуатацию.
  • Созданный документ – ссылка на документ "Состояние объектов ремонта".

Табличная часть «Состояния позже ввода в эксплуатацию»

Предназначена для вывода данных об объектах ремонта, у которых первое состояние введен позже даты ввода эксплуатацию.
  • № – порядковый номер строки.
  • Объект ремонта – наименование объекта ремонта.
  • Инвентарный номер – инвентарный номер объекта ремонт.
  • Заводской номер – номер изделия–объекта ремонта, установленный на заводе по номенклатуре объектов.
  • Типовой ОР – наименование типового объекта ремонта (ремонтной группы), в которую входит объект ремонта (отображается, если включена настройка "Использовать типовые объекты ремонта" в разделе Настройка параметров системы - Интерфейс и ввод документов - Оборудование и нормативы).
  • Положение – положение объекта ремонта в иерархии.
  • Есть нормативы – отметка о наличии у объекта ремонта нормативов (вкладка "Нормативы планирования - Нормативные ремонты" справочника "Объекты ремонта").
  • Статус в учете - статус объекта ремонта в учете (отображается, если включена настройка "Использовать документы принятия и списания оборудования" в разделе Настройка параметров системы - Интерфейс и ввод документов - Оборудование и нормативы).
  • Дата ввода в эксплуатацию - дата ввода в эксплуатацию объекта ремонта.
  • Дата первого состояния - дата первого введенного для объекта ремонта состояния.
  • Первое состояние - вид эксплуатации в первом введенном для объекта ремонта состояния.
  • Ввести состояние на - предлагаемая дата для ввода первого состояния объекта ремонта (дата равна дате ввода в эксплуатацию объекта ремонта).
  • Созданный документ – ссылка на документ "Состояние объектов ремонта".
