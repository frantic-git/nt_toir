Обработка «Управление плановой доступностью персонала»

Обработка предназначена для установки плановой доступности персонала. Плановая доступность персонала определяется графиками работ сотрудников с учетом выходных, отпусков, больничных и др. 


В форме выводятся следующие поля:

  • Организация — наименование организации, для сотрудников которой указывается доступность персонала.
  • Подразделение — наименование подразделения, для сотрудников которого указывается доступность персонала.
  • Период планирования — период, за который вводятся значения состояния доступности сотрудника.

С помощью кнопки [Обновить] можно переустановить отборы и очистить нижележащие разделы. 

Раздел «Плановая доступность персонала / Персонал»


С помощью кнопки [Добавить] можно добавить сотрудника.

С помощью кнопки [Заполнить сотрудников] можно автоматически заполнить табличную часть сотрудниками, которые трудятся в выбранной организации и выбранном подразделении.

  • Сотрудник — ФИО сотрудника организации, для которого указывается доступность.
  • Должность — должность сотрудника, для которого указывается доступность.
  • График работы — установленный график работы сотрудника.

Раздел «Плановая доступность персонала / Плановая доступность»

С помощью кнопки [Заполнить по штатному расписанию] можно автоматически заполнить доступность сотрудника/сотрудников по установленному для каждого из них расписанию.

С помощью кнопки [Редактировать ячейку расписания] можно точечно скорректировать доступность сотрудника в один конкретный день или некий период.

С помощью кнопки [Применить] можно сохранить изменения в документ "Плановая доступность персонала". До этого момента на вкладке "Плановая доступность персонала / Персонал" будет отображаться голубая плашка с надписью о наличии несохраненных изменений.

Раздел «Документы / Плановая доступность персонала»


Флаг Только проведенные позволяет отображать только проведенные документы. 

  • Номер — номер документа.
  • Дата — дата создания документа.
  • Организация — наименование организации, для сотрудников которой указывается доступность персонала.
  • Подразделение — наименование подразделения, для сотрудников которого указывается доступность персонала.
  • Начало планирования — дата начала периода, на который введен документ.
  • Окончание планирования — дата окончания периода, на который введен документ.

Флаг Обновлять при открытии позволяет автоматически обновлять данные в разделах при открытии формы.
