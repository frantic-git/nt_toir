Форма «Расписание работы сотрудника»

Форма предназначена для точечной корректировки доступности сотрудника, в случаях, когда штатное расписание оказывается неприменимо.


В форме выводятся следующие поля:

  • Дата графика — дата, на которую указывается доступность сотрудника.
    • С помощью кнопки [Установить до даты] можно установить данные из формы на несколько дней: от даты графика до выбранной. 
  • Сотрудник — ФИО сотрудника организации пользователя, для которого указывается доступность.
  • Состояние сотрудника — значение состояния доступности сотрудника.
  • Время работы – общее время работы сотрудника.

Раздел «Расписание работ»

Раздел отображается только для предопределенного состояния "Явка".

С помощью кнопки [Добавить] можно добавить строку расписания.

С помощью кнопки [Заполнить по графику] можно заполнить данные по графику работ подразделения.

  • Время начала — время начала действия состояния сотрудника.
  • Время окончания — время окончания действия состояния сотрудника.
  • Время работы — время работы сотрудника.
