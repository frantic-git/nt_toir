«Настройки ресурсное планирования работ»

Форма используется для установки настроек рабочего места «Ресурсное планирование работ подразделения».

Раздел «Ремонты оборудования»

 

Отображать обеспеченные — настройка, позволяющая отображать ремонты для которых подобран ремонтный персонал.
Отображать прошедшие — настройка, позволяющая отображать уже прошедшие ремонты.
Отображать ремонты без заявок — настройка, позволяющая отображать ремонты инициированные план-графиком ППР с отсутсвующими заявками.

Отображать завершенные — настройка, позволяющая отображать завершенные ремонты.

Отображать ремонты подчиненных подразделений — настройка, позволяющая отображать ремонты подчиненных подразделений при установленном отборе по подразделению исполнителю.

 

Фильтр Текущая дата – Дата актуальности позволяет задать точку отсчёта для отображения данных.

Плановый период позволяет выбрать длительность периода отображения данных относительно точки отсчета, выбранной выше. 

Раздел «Ресурс исполнителей»

 

Все сотрудники организации — выводить сотрудников всех подразделений организации в ресурсе исполнителей.


С помощью кнопки [Применить] данные сохраняются и применяются в открытой форме.

 

С помощью кнопки [Закрыть] можно выйти из формы, не сохраняя изменения и не меняя параметры отображения.
