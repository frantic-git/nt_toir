Обработка «Утверждение заказов на внутренние потребления»

Рабочее место предназначено для быстрой и эффективной работы технического директора по контролю расходов на ремонты и, в частности, количества и стоимости номенклатуры, закупаемой под заказы на внутренние потребления.

Отображение обработки зависит от функциональной опции "Использовать утверждение заказов на внутренние потребления", расположенной в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО. 

Группа «Отборы»

  • Период документов – отбор по дате в шапке документа "Заказ на внутреннее потребление".
  • Период потребности – отбор по реквизиту "Дата отгрузки" в шапке документа "Заказ на внутреннее потребление".
  • Организация – отбор по организации, в которой зарегистрирован документ "Заказ на внутреннее потребление".
  • Подразделение – отбор по подразделению, в котором зарегистрирован документ "Заказ на внутреннее потребление".
  • Ответственный – отбор по ответственному из документа "Заказ на внутреннее потребление".
  • Скрывать утвержденные – при установленном флаге не отображаются строки в статусе "Утвержден".
  • Скрывать отмененные – при установленном флаге не отображаются строки в статусе "Отменено/ Отклонен".
Кнопка [Обновить] предназначена для обновления информации во всем рабочем месте.
кнопка [Утвердить все] предназначена для утверждения всех позиций номенклатуры, отображающихся в рабочем месте.

Закладка «Номенклатура»

Табличная часть "Номенклатура"

  • Номенклатура - наименование номенклатуры.
  • Характеристика - наименование характеристики номенклатуры. Выводится, если установлена настройка системы в разделе Настройка параметров системы - Интерфейс и ввод документов - МТО: Использовать характеристики номенклатуры.
  • Количество, Ед. изм. - количество в базовой единице измерения номенклатуры.
  • Цена - цена номенклатуры на текущую дату из настройки "Тип цены для расчета плановой себестоимости ремонта" (раздел Настройка параметров системы - Общие).
  • Стоимость - стоимость номенклатуры. Рассчитывается как Цена * Количество.
  •  - отметка о возможности замены номенклатуры на аналоги.
  •  - отметка о том, что все или некоторые позиции заказов по данной номенклатуре отменены. Если отменены все позиции - красная пиктограмма, если только часть - серая.
  •  - отметка о том, что все или некоторые позиции заказов по данной номенклатуре утверждены. Если утверждены все позиции - зеленая пиктограмма, если только часть - серая.
Кнопка [Утвердить/ Отменить утверждение] предназначена для утверждения/отмены утверждения позиций номенклатуры.

Табличная часть "Дерево заказов и ремонтов" 

  • Ответственный - ответственный из документа "Заказ на внутреннее потребление".
  • Подразделение - подразделение из шапки документа "Заказ на внутреннее потребление".
  • Количество, Ед.изм. - количество и единица измерения из документа "Заказ на внутреннее потребление".
  • Стоимость - количество, переведенное в базовую единицу измерения, умноженное на цену из ТЧ "Номенклатура".
  • Дата потребности - реквизит "Дата отгрузки" из документа "Заказ на внутреннее потребление".
  • Крайний срок заказа - крайний срок заказа номенклатуры (зависит от схемы обеспечения номенклатуры. Рассчитывается по формуле: (Дата отгрузки – Срок покупки (в рабочих днях) от даты отгрузки) + 1 день на поступление товаров на склад).
  • Заказ - ссылка на документ "Заказ на внутреннее потребление".
  • Объект ремонта - наименование объекта ремонта*.
  • Вид ремонта - наименование вида ремонта*.
  • Дата начала - плановая дата начала ремонта*.
  • Документ-источник - ссылка на документ-источник*.
  •  - отметка о том, что для данной строки заказа были введены комментарии.
Кнопки  предназначены установки/снятия/инвертирования флажков.
Кнопка [Утвердить/ Отменить утверждение] предназначена для утверждения/отмены утверждения позиций номенклатуры. 
Кнопка [Вернуть с замечанием] предназначена для ввода замечания/комментария для выбранной позиции номенклатуры.
Кнопка [Отменить/ Снять отмену] предназначена для отмены/снятия отмены позиции номенклатуры.

  • Итого заказов на сумму - итог по колонке "Стоимость" из ТЧ "Номенклатура".
  • Утверждено - суммарная стоимость только утвержденных строк.
  • Не проверено - суммарная стоимость строк в статусе "На утверждение" и "Замечания исправлены".

Закладка «Заказы»

  • N - порядковый номер строки.
  • Заказ - ссылка на документ "Заказ на внутреннее потребление".
  • Дата потребности - реквизит "Дата отгрузки" из документа "Заказ на внутреннее потребление".
  • Ответственный - ответственный из документа "Заказ на внутреннее потребление".
  • Подразделение - подразделение из шапки документа "Заказ на внутреннее потребление".
  • Стоимость - суммарная стоимость всех строк заказа, кроме отмененных. 
  • Статус утверждения - реквизит "Статус утверждения" из шапки документа" Заказ на внутреннее потребление".
Кнопка [Утвердить/ Отменить утверждение] предназначена для утверждения документа "Заказ на внутреннее потребление".

* Если документ "Заказ на внутреннее потребление" создан вручную и не привязан к ремонтам, то поля "Объект ремонта", "Вид ремонта", "Дата начала", "Документ-источник" не заполняются.
