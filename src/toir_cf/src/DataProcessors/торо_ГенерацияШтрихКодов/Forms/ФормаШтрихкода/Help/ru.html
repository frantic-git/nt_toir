Форма «Генерация штрихкода»

Предназначена для создания штрихкода объекта ремонта и просмотра его графического изображения. Доступна по кнопке открытия поля Штрихкод из формы элемента справочника Объекты ремонта и формы обработки Генерация штрихкодов объектов ремонта.

Для генерации нового штрихкода нужно выбрать его тип и/или нажать Сгенерировать.
