Обработка «Генерация штрихкодов объектов ремонта»

Обработка предназначена для массового создания штрихкодов объектов ремонта. Для этого нужно отметить флажками строки дерева с объектами ремонта (вручную, или с помощью кнопок в командной панели дерева), для которых нужно сгенерировать штрихкоды, выбрать тип новых штрихкодов и нажать Сгенерировать.

Также в командной панели дерева расположены команды  Развернуть все и  Свернуть все для управления группировками и  Настройка иерархии для выбора структуры иерархии объектов ремонта, отображаемой на форме.

Для сохранения новых штрихкодов можно воспользоваться командой  Записать (или Записать и закрыть). Команда  Печать формирует печатные формы с графическим изображением сгенерированных штрихкодов.
