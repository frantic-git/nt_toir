Обработка «Подбор номенклатуры»

Обработка используется для подбора номенклатуры в табличные части документов. 

В форме выводятся следующие поля:

  • Склад – отбор по складу, к которому относится номенклатура.
  • Отобразить всю номенклатуру – настройка, позволяющая отобразить при подборе в документа ремонтов всю номенклатуру, отключив отбор по списку запчастей ОР.
  • Использовать серии – настройка, позволяющая использовать серии номенклатуры при подборе в документы МТО. Отображается при включенной функциональной опции "Использовать серии номенклатуры". 
  • Фильтр – ограничение области поиска товаров: по иерархии номенклатуры или по ее виду. В соответствии с выбранным вариантом отбора будет показан список групп номенклатуры или список видов номенклатуры.

Табличная часть «Список номенклатуры»

Содержит список позиций номенклатуры, принадлежащих группе или виду номенклатуры, в зависимости от выбранного фильтра. Подбор номенклатуры осуществляется в этой табличной части. Чтобы выбрать номенклатуру, нужно дважды кликнуть на ней левой кнопкой мыши либо нажать клавишу Enter, затем ввести количество номенклатуры и нажать кнопку OK.
  • Артикул – артикул номенклатуры.
  • Номенклатура – наименование номенклатуры.
  • Серия – наименование серии номенклатуры. Отображается при включенной функциональной опции "Использовать серии номенклатуры". 
  • Характеристика – наименование характеристики номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры". 
  • Единица хранения – единица хранения номенклатуры.
  • В наличии – количество единиц номенклатуры в наличии.

Группа «Подобранная номенклатура»

Выбранная в табличной части "Список номенклатуры" номенклатура отображается в табличной части группы "Подобранная номенклатура". Заголовок группы отображает количество подобранных позиций номенклатуры (не общее количество единиц номенклатуры). 

  • N – порядковый номер записи в списке.
  • Номенклатура – наименование подобранной номенклатуры.
  • Характеристика – наименование характеристики подобранной номенклатуры. Отображается при включенной функциональной опции "Использовать характеристики номенклатуры". 
  • Серия – наименование серии номенклатуры. Отображается при включенной функциональной опции "Использовать серии номенклатуры". 
  • Единица измерения – единица, в которых измеряется количество подобранной номенклатуры.
  • Количество – количество подобранной номенклатуры.
  • Вид цены – вид цены подобранной номенклатуры.
  • Цена – цена подобранной номенклатуры.
  • Сумма – стоимость подобранного количества номенклатуры.
  • Склад – наименование склада, с которого подбирается номенклатура. 

Группа «Остатки по складам»

Содержит информацию по остаткам на складах выбранной в табличной части "Список номенклатуры" номенклатуры.

  • Склад – наименование склада.
  • Доступно – количество номенклатуры на складе, доступной к подбору.
Кнопка [Завершить подбор] позволяет завершить подбор номенклатуры и перенести ее в документ.
