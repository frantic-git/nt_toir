Рабочее место диспетчера

Рабочее место предназначено для выполнения обязанностей диспетчера.

На форме присутствуют следующие поля для отбора:

  • Тип поиска – содержит тип поля для отбора.
  • Значение поиска – данные поля для отбора.

Кнопка [Журнал дефектов] позволяет открыть журнал дефектов.

Кнопка [Отчет] позволяет перейти к формам отчетов "Выявленные дефекты" и "Внешние основания".

Кнопка [Документ] позволяет перейти к форме создания документа для выявленных дефектов и внешних оснований.

Табличная часть «Дерево»

  • Наименование – наименование объекта ремонта или группы.
  • Код – код объекта ремонта или группы.
  • Инвентарный № – инвентарный номер объекта ремонта.

Кнопка [Журнал объекта ремонта] открывает журнал объекта ремонта.

Кнопка [Закрытие заявок и ремонтов] открывает обработку "Закрытие заявок и ремонтов".

Группа «Данные текущего объекта ремонта»

  • Подразделение – подразделение-владелец объекта ремонта.
  • Входит в состав – группа, в которую входит объект ремонта.
  • Наименование – наименование объекта ремонта.
  • Местонахождение – расположение объекта ремонта.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • Заводской № – заводской номер объекта ремонта.
  • Технологический № – технологический номер объекта ремонта

Кнопка [Зарегистрировать] позволяет зарегистрировать документ "Выявленный дефект" или документ "Внешнее основание для работ".

Кнопка [Новый] очищает заполненные поля, используемые при создании документа.

Кнопка [Печать] позволяет распечатать документы "Акт дефектации", "Дефектная ведомость", "Дефектная ведомость (ликвидация ОР)" и "Акт технического осмотра", основываясь на введенных данных.

Группа «Запланированные ремонты»

Содержит информацию о запланированных ремонтах объекта ремонта.

Закладка «Плановые ремонты»

Содержит информацию о ближайших десяти ремонтах выбранного объекта ремонта из план-графика ППР.
  • Плановая дата начала – плановая дата начала ремонта из план-графика ППР.
  • Плановая дата окончания – плановая дата окончания ремонта из план-графика ППР.
  • Вид ремонта – вид планового ремонта.
  • Основание – дата и номер документа-основания.
  • Документ плановых дат – дата и номер документа плановых дат.

Закладка «Выявленные дефекты»

Содержит информацию о выявленных в выбранном объекте ремонта дефектах.
  • Документ – дата и номер документа "Выявленный дефект".
  • Отказавший элемент – наименование отказавшего элемента объекта ремонта.
  • Дата обнаружения – дата обнаружения дефекта.
  • Вид дефекта – вид обнаруженного дефекта.
  • Описание дефекта – текстовое описание выявленного дефекта.
  • Статус – статус, в котором находится выявленный дефект. Зарегистрирован – документ создан; Запланировано устранение – на основании создан документ "Смета ремонта (Заявка на ремонт)"; Устраняется – на основании создан документ "Наряд на выполнение ремонтных работ"; Устранен – на основании создан документ "Акт о выполнении этапа работ"; Отклонен – дефект отменен через обработку "Закрытие заявок и ремонтов".

Закладка «Внешние основания»

Содержит информацию о внешних основаниях, зарегистрированных для выбранного объекта ремонта.
  • Документ – дата и номер документа "Внешнее основание".
  • Описание – описание типового дефекта из внешнего основания.
  • Срок – дата окончания работ по внешнему основанию.
  • Статус – статус, в котором находится внешнее основание для работ. Зарегистрирован – документ создан; Запланировано устранение – на основании создан документ "Смета ремонта (Заявка на ремонт)"; Устраняется – на основании создан документ "Наряд на выполнение ремонтных работ"; Устранен – создан документ "Акт о выполнении этапа работ".

Закладка «Выявленный дефект»

  • Номер – номер документа.
  • Дата – дата документа.
  • Зарегистрированный документ – после регистрации документа "Выявленный документ" здесь появится ссылка на него.
  • Подразделение-исполнитель – подразделение-исполнитель для объекта ремонта.
  • Контактное лицо – контактное лицо для обращения.
  • Инициатор – инициатор создания документа.
  • Дата обнаружения – дата обнаружения дефекта.
  • Подразделение – подразделение, в котором зарегистрирован дефект. 
  • Организация – организация объекта ремонта.
  • Отказавший элемент – отказавший элемент объекта ремонта.
  • Вид дефекта – вид дефекта.
  • Типовой дефект – типовой дефект объекта ремонта.
  • Непосредственная причина дефекта –непосредственная причина возникновения дефекта.
  • Критичность дефекта – критичность дефекта.
  • Дефект выявлен в процессе – работы, при которых выявлен дефект.

Группа «Дополнительная информация о дефекте»

  • Описание дефекта – описание выявленного дефекта.
  • Последствие дефекта – последствия выявленного дефекта.
  • Комментарий к направлению на устранение – комментарий.

Закладка «Внешнее основание для работ»

  • Номер – номер документа.
  • Дата – дата документа.
  • Организация – организация объекта ремонта.
  • Подразделение – подразделение, в котором зарегистрировано внешнее основание. 
  • Плановая дата ремонта – плановая дата осуществления ремонтных работ.
  • Ответственный – ФИО сотрудника, ответственного за выполнение предписанных мероприятий.
  • Исполнитель – подразделение-исполнитель ремонтных работ.
  • Типовой дефект – типовой дефект объекта ремонта.
  • Описание – описание дефекта.

Табличная часть «Рекомендуемые операции» 

  • Ремонтная работа – ссылка на ремонтные работы.
  • Количество – количество ремонтных работ.
  • Ед. Изм. – единицы измерения.

Кнопка [Добавить тех операцию] позволяет добавить технологическую операцию из меню выбора.

Кнопка [Подбор] позволяет подобрать несколько технологических операций из меню выбора.
