Рабочее место «Настройка показателей эксплуатации объектов ремонта»

Предназначена для быстрой и эффективной работы технического специалиста по настройке показателей эксплуатации объектов ремонта.

Группа «Отборы»

  • Организация – отбор по организации-владельцу объекта ремонта.
  • Направление – отбор по направлению объекта ремонта.
  • Подразделение – отбор по подразделению-владельцу объекта ремонта.
  • Типовой ОР – отбор по типовому объекту ремонта (ремонтной группе).
  • Поле поиска – поле для указания параметра, по которому будет производиться поиск.
  • Критичность – отбор по критичности объекта ремонта.
  • Содержит – поле для указания значения параметра поиска.
  • Показатели – отбор по наполненности показателей у объекта ремонта.

Табличная часть «Дерево»

  • Наименование – наименование объекта ремонта или группы объектов ремонта.
  • Код – код объекта ремонта или группы.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • К.П. – признак наличия данных по контролируемым показателям у объекта ремонта.
  • П.Н. – признак наличия данных по показателям наработки у объекта ремонта.

Группа «Данные текущего объекта ремонта»

Предназначена для вывода данных об объекте, выбранном в табличной части "Дерево".
  • Наименование – наименование объекта ремонта.
  • Направление – наименование направления объекта ремонта.
  • Критичность – наименование критичности объекта ремонта.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • Технологический № – технологический номер изделия–объекта ремонта.
  • Типовой ОР – наименование типового объекта ремонта (ремонтной группы).
  • Состояние ОР – состояние объекта ремонта (зависит от документа "Состояния объектов ремонта").
  • Подразделение – наименование подразделения-владельца объекта ремонта.
  • Заводской № – номер изделия–объекта ремонта, установленный на заводе по номенклатуре объектов.
  • Причина простоя – причина простоя (заполняется, если состояние ОР – простой).
Кнопка [Записать] позволяет записать в систему введенные данные.

Группа «Показатели»

Предназначена для ввода контролируемых показателей и настройки параметров уведомлений и регистрации дефекта.

Табличная часть «Контролируемые показатели»

  • Показатель – название измеряемого показателя.
  • Ед. изм. – единица измерения измеряемого показателя.
  • Из ОР – признак, при установке которого становится доступно редактирование допустимых границ показателя.
  • Допустимый минимум – минимальное допустимое значение контролируемого показателя.
  • Допустимый максимум – максимальное допустимое значение контролируемого показателя
  • Критический минимум – минимальное критическое значение контролируемого показателя.
  • Критический максимум – максимальное критическое значение контролируемого показателя.

Табличная часть «Параметры уведомлений и регистрации дефектов»

Доступна при включенной функциональной опции "Использовать ремонты по состоянию".
  • Вид сравнения – вид сравнения с показателем объекта ремонта (может принимать значения: больше, меньше, в интервале и т.д.).
  • Вид значения – вид значения сравнения. Может принимать значение допустимых границ или произвольное.
  • Параметры уведомления – шаблон, по которому для пользователя формируются уведомления.
  • Параметры ввода дефекта – шаблон, по которому автоматически формируется документ "Выявленные дефекты".

Группа «Наработка»

Предназначена для ввода параметров наработки и их максимальных значений.

Табличная часть «Параметры наработки»

  • Дата введения – дата введения показателя плановой наработки (по умолчанию подставляется дата ввода в эксплуатацию объекта ремонта)
  • Показатель наработки – наименование показателя, по которому фиксируется наработка текущего объекта ремонта.
  • Основной – признак, при установке которого становится доступно редактирование параметра наработки (для параметров наработки, унаследованных от типовых объектов ремонта).
  • Ед. изм. – единица, в которой измеряется наработка объекта ремонта для текущего показателя.
  • Период наработки – промежуток времени, на который вводится плановое значение наработки. Значение выбирается из выпадающего списка и может содержать одно из предопределенных значений: День, Неделя, Декада, Месяц, Квартал, Полугодие, Год.
  • Плановая наработка – значение, которое будет использоваться для расчета наработки объекта ремонта в случае отсутствия фактических значений в журнале наработки.
  • КИО – коэффициент учета плановой наработки. Коэффициент учитывает долю аварийных ремонтов при планировании по наработке.
  • Ресурс гарантии – значение наработки ресурса гарантии (заполняется при условии гарантии с проверкой наработки (вкладка "Гарантийные условия" карточки объекта ремонта).
Кнопка [Заполнить] позволяет заполнить данные показателя из типового объекта ремонта. Данные можно заполнить по одному, нескольким или всем показателям.

Табличная часть «Максимальные значения»

  • Максимальное значение – максимальное значение показателя наработки, при достижении которого счетчик наработки обнуляется.
  • Назначенный ресурс – суммарная наработка объекта, при достижении которой эксплуатация должна быть прекращена независимо от его состояния
  • Остаточный ресурс – суммарная наработка оборудования от момента проведения контроля его технического состояния, до перехода в предельное состояние.
