Рабочее место «Адресация заявок»

Рабочее место предназначено для регистрации и распределения по исполнителям документов "Смета ремонта (Заявка на ремонт)".

Кнопка [Вариант отображения] позволяет настроить вариант отображения списка документов для формы: без заявок, с заявками, все.

Кнопка [Открыть РМ диспетчера] открывает рабочее место диспетчера.

На форме присутствуют следующие поля для отбора:

  • Период – отметка и поля для ввода дат, по которым будут отобраны ремонты.
  • Выводить данные по – выбор документов-оснований, по которым будут выводиться ремонты в табличной части.
Кнопка [Открыть документ-основание] позволяет открыть документ-основание для выбранной строки.
Кнопка [Ввести заявку] позволяет ввести заявку на основании выбранного ремонта.
Кнопка [Обновить] обновляет табличную часть.

Табличная часть «Ремонты»

  • Документ-основание – ссылка на документ-основание.
  • Объект ремонта – наименование объекта ремонта.
  • Вид ремонта – наименование вида ремонта из документа-основания.
  • Дата начала – дата начала выполнения ремонта.
  • Дата окончания – дата окончания выполнения ремонта.
  • Описание – дополнительное описание.
  • Статус ремонта – статус ремонта (поле "Статус ремонта" ТЧ "Ремонтные работы" из документа "Смета ремонта (Заявка на ремонт)").
  • Исполнитель – наименование исполнителя (поле "Подразделение-исполнитель" ТЧ "Объекты ремонта" из документа "Выявленные дефекты"; поле "Исполнитель" ТЧ "Перечень внешних оснований для работ" из документа "Внешнее основание для работ"; поле "Исполнитель" ТЧ "План ремонтов" из документа "План-график ППР"; при наличии документа "Смета ремонта (Заявка на ремонт) поле "Исполнитель" из ТЧ "Исполнители - По ремонту").
  • Бригада/Договор – наименование бригады для хозспособа, наименование договора - при подрядном способе (поле "Договор/ Бригада" ТЧ "По ремонту" из документа "Смета ремонта (Заявка на ремонт").

В вариантах отображения "С заявками" и "Все" к реквизитам рабочего места добавляются:

  • Статус – выбор статуса ремонта для отбора по нему ремонтов в табличной части.
  • Подразделение – выбор подразделения-исполнителя для отбора по нему ремонтов в табличной части.
  • Бригада – выбор бригады для отбора документов по ней в табличной части.
  • Контрагент – выбор контрагента для отбора по нему ремонтов в табличной части.
  • Договор контрагента – выбор договора для отбора по нему ремонтов в табличной части. 
К реквизитам табличной части добавляются реквизиты:

  • Введена заявка – наличие введенной заявки на ремонт.
  • Заявка на ремонт – ссылка на документ "Смета ремонта (Заявка на ремонт)".
