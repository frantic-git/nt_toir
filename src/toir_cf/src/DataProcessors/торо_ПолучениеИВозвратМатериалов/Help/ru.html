Рабочее место «Получение и возврат материалов»

Рабочее место предназначено для анализа обеспеченности материалами запланированных ремонтов (данные берутся из документов "План-график ППР", "Смета ремонта (Заявка на ремонт)").

Кнопка [Переместить излишки на другой склад] позволяет сформировать документ "Перемещение товаров" для передачи материалов со склада, указанного в шапке обработки, на другой склад (указывается в формируемом документе).

Кнопка [Устранить дефицит заказами] позволяет сформировать документы "Заказ на внутреннее потребление" на недостающее количество (покрыть планируемый дефицит). При выборе нескольких позиций номенклатуры, по которым имеется излишек, пользователь может сформировать документ сразу на необходимое количество.

На форме присутствуют следующие поля для отбора:

  • Подразделение - подразделение-исполнитель ремонта.
  • Склад - склад, по которому выполняется проверка остатков материалов и излишка (дефицита) ТМЦ. Если склад не указан, показываются остатки на всех складах.
  • Период - период, по которому выполняется анализ запланированных ремонтов.

В табличной части Материалы содержатся следующие поля:

  • Номенклатура - позиция номенклатуры, требуемая для выполнения ремонтов.
  • Ед. изм. - единица измерения номенклатуры.
  • Остаток на складе - количество номенклатуры на складе, указанном в отборе в верхней части обработки. Если склад не указан, показываются остатки по всем складам.
  • Остаток всего - общее количество номенклатуры по всем складам.
  • Текущий излишек (дефицит) - текущий излишек (со знаком плюс) или недостаток (со знаком минус) требуемой для выполнения ремонтов номенклатуры. Количество номенклатуры рассчитывается из потребностей в материалах по ремонтам, выполнение которых уже начато, и остатка на выбранном складе. После ввода документа "Заказ на внутреннее потребление" потребность считается обеспеченной.
  • Планируемый излишек (дефицит) - планируемый излишек (со знаком плюс) или недостаток (со знаком минус) требуемой для выполнения ремонтов номенклатуры. Рассчитывается из потребностей в материалах по всем невыполненным ремонтам, остатка на выбранном складе и заказанных документом "Заказ на внутреннее потребление" материалов.

При двойном клике по строке с номенклатурой становится доступно отдельное окно с расшифровкой по двум типам данных для выбора:

  • Список ремонтов, где требуется выбранная номенклатура. В формируемом списке отображаются документы плана ремонтов "План-график ППР" и "Смета ремонта (Заявка на ремонт)", из которых возникает потребность в выбранном материале (запчасти). В списке отражается информация о том, введены ли документы "Заказ на внутреннее потребление", на все ли материалы введены документы "Заказ на внутреннее потребление".
  • Список документов "Заказ на внутреннее потребление", в которых присутствует выбранный материал (запчасть).
