Обработка «Рабочее место технолога»

Обработка предназначена для:

  • Просмотра структуры функциональных мест и установленного на них оборудования.
  • Просмотра ограничений, установленных для функциональных мест.
  • Проверки соответствия установленного оборудования ограничениям с возможностью снять или установить оборудование.
  • Добавления функциональных мест.

На форме выводятся следующие поля:

  • Тип поиска - содержит тип поля для отбора.
  • Содержит - данные поля для отбора.

В табличной части Дерево содержатся следующие поля:

  • Наименование - наименование объекта ремонта или группы.
  • Код - код объекта ремонта или группы.
  • Инвентарный № - инвентарный номер объекта ремонта.

Кнопка [Добавить ФМ] предназначена для добавления функционального места (нового или из другой иерархии).

Кнопка [Проверить соответствие ограничениям] предназначена для проверки выделенного объекта ремонта на соответствие. Несоответствующие объекты ремонта попадают в нижнюю табличную часть.

В табличной части Несоответствующие объекты ремонта содержатся следующие поля:

  • Объект ремонта - ссылка на объект ремонта.

Закладка «Ограничения на характеристики»

  • Ограничение - ссылка на ограничения на характеристики оборудования.

Закладка «Список характеристик»

  • Характеристика - ссылка на план видов характеристик объекта ремонта.
  • Значение - значение характеристики объекта ремонта.
