Обработка «Установка оценки РМ плановых ремонтов»

Обработка предназначена для установки оценок риск-менеджмента плановых ремонтов.

В форме выводятся следующие поля:

  • План-график ремонта – документ "План-график ППР", для которого необходимо установить оценки РМ плановых ремонтов.

Группа «Отборы»

Позволяет накладывать отборы на ремонты, которые будут выводиться в табличную часть.
  • Владелец ОР – отбор по контрагенту или организации, которой принадлежит объект ремонта.
  • Подразделение – отбор по подразделению объекта ремонта.
  • Вид ремонта – отбор по виду ремонта.
  • Способ выполнения – отбор по способу выполнения ремонта: хозспособ или подрядный.
  • Объект ремонта – отбор по объекту ремонта.
  • Направление – отбор по направлению объекта ремонта.
  • Сумма от... до: – отбор по сумме ремонтных работ.

Табличная часть «План ремонтов»

Содержит список ремонтов, по которым оценивается вероятность выхода из строя и возможный ущерб. Поля с оценками заполняются экспертом. 
  • Вид ремонтных работ – вид планового ремонта.
  • Объект ремонтных работ – наименование объекта ремонта.
  • Дата начала ремонтных работ – дата начала ремонта.
  • Дата окончания ремонтных работ – дата окончания ремонта.
  • Исполнитель – наименование подразделения или контрагента-исполнителя ремонта.
  • Способ выполнения – способ выполнения ремонта: хозспособ или подрядный. 
Поля "Вероятность выхода из строя" и "Ущерб" заполняются вручную экспертом. Остальные поля заполняются из выбранного план-графика ППР с учетом отборов по кнопке [Заполнить]. 
При установке нового отбора табличную часть можно очистить полностью и заполнить заново (заполнить строки по отбору), а можно добавить ремонты, соответствующие новому отбору, к отобранным ранее (добавить строки по отбору).
