Обработка «Контроль результатов паспортизации»

Обработка предназначена для контроля заполнения справочников "Объекты ремонта", "Типовые объекты ремонта" и "Технологические карты".

Кнопка [Обновить] обновляет табличные части обработки.

Закладка «Объекты ремонта»

Отображает заполненность элементов справочника "Объекты ремонта".

На закладке выводятся следующие поля:

  • Организация – наименование организации объекта ремонта.
  • Подразделение – наименование подразделения объекта ремонта.
  • Направление – наименование направления объекта ремонта.

Подзакладка «Без связи с типовыми объектами ремонта»

Предназначена для контроля заполнения элементов справочника "Объекты ремонта", для которых не указан типовой объект ремонта.

На подзакладке выводятся следующие поля:

  • Код – код объекта ремонта.
  • Объект ремонта – наименование объекта ремонта.
  • Заводской № – заводской номер объекта ремонта.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • Технологический № – технологический номер объекта ремонта.

Подзакладка «Без запчастей»

Предназначена для контроля заполнения элементов справочника "Объекты ремонта", у которых нет запчастей. 

На подзакладке выводятся следующие поля:

  • Код – код объекта ремонта.
  • Объект ремонта – наименование объекта ремонта.
  • Заводской № – заводской номер объекта ремонта.
  • Инвентарный № – инвентарный номер объекта ремонта.
  • Технологический № – технологический номер объекта ремонта.

Подзакладка «Нормативы ремонтов»

Предназначена для контроля заполнения подзакладки "Нормативные ремонты" элементов справочника "Объекты ремонта". Состоит из трех разделов: "Без нормативных ремонтов", "Без предшествующих ремонтов", "Без замещаемых ремонтов", в каждом из которых выводятся следующие поля:
  • Код – код объекта ремонта.
  • Объект ремонта – наименование объекта ремонта.
  • Направление – направление объекта ремонта.

Закладка «Типовые объекты ремонта»

Отображает заполненность элементов справочника "Типовые объекты ремонта".

На закладке выводятся следующие поля:
  • Направление – направление типового объекта ремонта.

Подзакладка «Без связи с объектами ремонта»

Отображает типовые объекты ремонта, которые не указаны как типовые ни для одного объекта ремонта.

На подзакладке выводятся следующие поля:
  • Код – код типового объекта ремонта.
  • Типовой ОР – наименование типового объекта ремонта.
  • Направление – направление типового объекта ремонта.

Подзакладка «Без запчастей»

Отображает типовые объекты ремонта, у которых нет запчастей.

На подзакладке выводятся следующие поля:
  • Код – код типового объекта ремонта.
  • Типовой объект ремонта – наименование типового объекта ремонта.
  • Направление – направление типового объекта ремонта.

Подзакладка «Нормативы ремонтов»

Предназначена для контроля заполнения подзакладки "Нормативные ремонты" элементов справочника "Типовые объекты ремонта". Состоит из трех разделов: "Без нормативных ремонтов", "Без предшествующих ремонтов", "Без замещаемых ремонтов", в каждом из которых выводятся следующие поля:

Закладка «Технологические карты»

Отображает заполненность элементов справочника "Технологические карты".

Подзакладка «Связи с объектами»

Предназначена для отображения технологических карт, которые не связаны с типовыми объектами ремонта либо с объектами ремонта. Состоит из двух разделов: "Не связанные с типовыми объектами ремонта" и "Не связанные с объектами ремонта", в каждом из которых выводятся следующие поля:
  • Код – код технологической карты.
  • Технологическая карта – наименование технологической карты.

Подзакладка «Затраты»

Предназначена для отображения технологических карт, у которых не заполнена закладка "Материальные затраты" либо закладка "Трудовые затраты". Состоит из двух разделов: "Без материальных затрат" и "Без трудовых затрат", в каждом из которых выводятся следующие поля:
  • Код – код технологической карты.
  • Технологическая карта – наименование технологической карты.
