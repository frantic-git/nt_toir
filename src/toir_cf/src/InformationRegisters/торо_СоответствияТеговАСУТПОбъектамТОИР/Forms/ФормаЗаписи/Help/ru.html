Регистр сведений «Соответствие тегов объектам ТОИР»

 

В форме записи регистра выводятся следующие поля:

  • Наименование ОР - наименование объекта ремонта.
  • Тег - произвольная строка для установления соответствия.
  • Подключение - наименование элемента из справочника «Подключение».
  • Объект ремонта - наименование элемента из справочника «Объекты ремонта».
  • Тип сигнала - наименование элемента из перечисления «типы сигналов АСУ ТП».
    • Если тип сигнала Состояние, то настройках сигнала могут быть указаны Состояние из справочника «Виды эксплуатации» и Причина простоя из справочника «Причины простоя оборудования».
    • Если тип сигнала Показатель, то настройках сигнала может быть указан Показатель из плана видов характеристик «Измеряемые показатели объектов ремонта».
    • Если тип сигнала Наработка, то настройках сигнала может быть указан Параметр наработки из справочника «Параметры выработки ОР».
    • Если тип сигнала Дефект, то настройках сигнала может быть указан Типовой дефект из справочника «Типовые дефекты оборудования».
    • Если тип сигнала KPI, то настройках сигнала может быть указан Показатель KPI из справочника «Показатели KPI».
  • Отключить - при установке этого флага данные настройки будут отключены.
  • Ответственный - наименование элемента из справочника «Пользователи».
  • Подразделение - наименование элемента из справочника «Структура предприятия».
  • Организация - наименование элемента из справочника «Организации».
