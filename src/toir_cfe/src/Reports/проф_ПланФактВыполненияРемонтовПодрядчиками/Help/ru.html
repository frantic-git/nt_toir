Отчет «План-факт выполнения ремонтов подрядчиками»

Отчет предназначен для получения информации о запланированных и фактических датах начала и окончания ремонтных работ, выполняемых подрядчиками. Информация в отчете выводится по каждому подрядчику с перечислением объектов ремонта, по которым запланированы работы.

Открывается по команде План-факт выполнения ремонтов подрядчиками в списке Отчеты по нарядам и работам подсистемы Наряды и работы.

Параметры отчета
Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. В отчете можно настроить отборы путем установки значений параметров отчета:
  • Период отчета - отбор плановых/внеплановых ремонтов по плановым датам;
  • Структура иерархии - отбор, позволяющий вывести полный путь к объекту ремонта в зависимости от выбранной иерархии. По умолчанию заполняется из пользовательских настроек (настройка "Основная структура иерархии");
  • Учитывать План-графики ППР - настройка, позволяющая учитывать ремонты из документов "План-график ППР";
  • Учитывать сметы (заявки на ремонт) по дефектам - настройка, позволяющая учитывать ремонты из документов "Выявленные дефекты";
  • Учитывать сметы (заявки на ремонт) по внешним основаниям - настройка, позволяющая учитывать ремонты из документов "Внешнее основание для работ";
  • Учитывать регламентные мероприятия - настройка, позволяющая учитывать ремонты из документов "График регламентных мероприятий".
Варианты отчета
С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Доступно два предопределенных варианта отчета:
  1. План-факт выполнения ремонтов подрядчиками (по исполнителям) - открывается по умолчанию. В этом варианте отчета данные в строках группируются по подрядчикам, выполняющим плановые ремонты. Фиксация плановых дат осуществляется документом "План-график ППР" и "График регламентных мероприятий",  фактических дат - документом "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия" .
  • Исполнитель - подрядчик, выполняющий работы (поле "Исполнитель" из ТЧ ремонтов документов "План-график ППР" и "График регламентных мероприятий");
  • Объект ремонта - наименование объекта ремонта;
  • Вид ремонта - вид ремонта (поле "Вид ремонта" из документа "План-график ППР" и поле "Вид мероприятия" из документа "График регламентных мероприятий");
  • Документ-основание - документы, на основании которых создаются документы плана;
  • Плановая дата начала - планируемая дата начала ремонтных работ (поле "Дата начала" из документа "Смета ремонта (Заявка на ремонт)" ; поле "Период" из документов "План-график ППР" и "График регламентных мероприятий");
  • Плановая дата окончания - планируемая дата окончания ремонтных работ (поле "Дата окончания" из документа "Смета ремонта (Заявка на ремонт"); поле "Период" из документов "План-график ППР" и "График регламентных мероприятий");
  • Фактическая дата начала - фактическая дата начала ремонтных работ (поле "Дата начала" из документов "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия");
  • Фактическая дата окончания - фактическая дата окончания ремонтных работ (поле "Дата окончания" из документов "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия" (с завершающими флагами);
  • Акт - ссылка на документы "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия";
  • Статус работы подрядчика - работе подрядчика может быть присвоено четыре статуса:
    • Привлечение данного подрядчика не планировалась (синим) - с этим контрагентом есть акт, но его не было в планах;
    • Подрядчик не выполнил работы (красным) - по контрагенту есть план, но нет акта;
    • Подрядчик не выбран (красным) - в ППР указан подрядный способ, но контрагент не заполнен и нет контрагентов в заявках и в актах;
    • Плановое выполнение работ подрядчиком (зеленым) - по контрагенту есть и план и факт.
    • Ремонт завершен - присваиваемый статус ремонта:
      • Нет - начальный статус для всех ремонтов;
      • Да - статус ремонта, который устанавливается после проведения документов "Акт о выполнении этапа работ" и "Акт о выполнении регламентного мероприятия"(с завершающими флагами).
    1. План-факт выполнения ремонтов подрядчиками (по ремонтам) - в этом варианте отчета данные в строках группируются по объектам ремонта, на которых подрядчики выполняли плановые ремонты. 
        Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
